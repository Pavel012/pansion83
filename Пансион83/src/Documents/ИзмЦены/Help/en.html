Документ "Приказ об изменении диапазонов признаков цены"

Назначение

В приказах об изменении диапазонов признаков цены указываются значения диапазонов процента загрузки или продолжительности проживания и их связи с признаками цены, которые будут использоваться при определении стоимости услуг начисляемых по тарифу при бронировании и проживании. Дата и время документа задают момент времени, когда приказ начинает действовать. Программа определяет, какие цены применить к каждой дате проживания гостя, путем поиска действующего на эту дату приказа об изменении диапазонов.

Значения диапазонов всегда задаются для гостиницы и типа признака цены. Эти параметры обязательны для ввода.

Относится к подсистеме: «Управление взаиморасчетами».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Гостиница. Ссылается на справочник «Гостиницы».
  • Тип признака цены. Ссылается на перечисление «Типы признаков цены». Обязательное для заполнения поле. Возможно заполнение диапазонов либо по продолжительности проживания гостей либо по значениям процента загрузки гостиницы в момент бронирования/размещения гостя.
  • Примечания. Тип значения: «Строка». Произвольные примечания к документу.
  • Автор. Ссылается на справочник «Сотрудники». Сотрудник, который создал документ.

Табличная часть «Диапазоны признака цены»

  • Признак цены. Ссылается на справочник «Признаки цены». Задает значение признака цены для указанного дипазона значений показателя загрузки гостиницы или продолжительности проживания гостя. Именно для этого значения признака цены программа будет искать приказ об изменении цен в алгоритме расчета состава и стоимости услуг начисляемых по тарифу проживания. 
  • Начальное значение дипазона. Тип значения: «Число». Начальное значение диапазона действия признака цены. Значение проверяется по условию >=.
  • Конечное значение дипазона. Тип значения: «Число». Конечное значение диапазона действия признака цены. Значение проверяется по условию <=.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма журнала приказов. Используется для отображения списка или выбора документов данного типа.

Способ вызова

Вызывается из меню "Взаиморасчеты/Справочники/Приказ об изменении диапазонов признаков цены" или из главного меню программы по кнопке "Еще../Закладка "2"/Тарифы", кнопка "Перейти" в панели инструментов списка тарифов.

Движения

  • «Диапазоны признаков цены».
