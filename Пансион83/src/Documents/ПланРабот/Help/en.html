Документ "План работ"

Назначение

Документ используется для ведения плана работ сотрудников (обычно горничных) на день. Документ позволяет сформировать список работ, которые должны быть выполнены в номерах и позволяет для каждой работы назначить исполнителя. Исполнителей работ можно назначать как вручную, так и автоматически по одному из 3-х алгоритмов. Порядок работы с документом следующий:

  1. Утром создается новый документ и в нем нажимается кнопка "Заполнить". По кнопке табличная часть документа заполняется списком всех номеров гостиницы и для каждого номера программа, если нужно, указывает работу, которая в этом номере должна быть выполнена. Состав работ настраивается на закладке "Параметры" и обычно заполняется автоматически из параметров карточки гостиницы.
  2. Открывается закладка "Сотрудники" и в ней нажимается кнопка "Заполнить". В табличную часть добавляются горничные, которые должны были выйти на работу по графикам работы. Если графики работы сотрудников не заполнены, то сотрудников в текущей смене можно добавить в документ вручную, выбирая их из справочника сорудников. В справочник сотрудников можно добавлять не только пользователей программы, но и сотрудников у которых к программе доступа нет. Для добавления новых сотрудников в справочнике сотрудники можно использовать кнопки "Добавить" и "Скопировать". В качестве кодов сотрудников добавленных в справочник вручную рекомендуется указывать их инициалы, например, для сотрудника Иванов Иван Иванович код сотрудника в справочнике может быть "ИИИ". В качестве наименования сотрудника рекомендуется указывать его фамилию с инициалами, т.е. Иванов И.И. в примере выше. Кроме кода и наименования у сотрудника должны быть указаны гостиница, отдел и должность. При назначении работ сотрудникам программа учитывает порядок следования сотрудников в списке на второй закладке. Рекомендуется выставить порядок сотрудников так, чтобы он примерно соответствовал порядку этажей в гостинице.
  3. После того как в смене определен список горничных, можно в строках работ указывать сотрудников, которые эти работы будут выполнять. Горничные указываются в колонке "Сотрудник", выбираются из списка доступных в смене. Назначать сотрудников можно вручную, можно автоматически по одному из 3-х реализованных алгоритмов. Алгоритм автоматического назначения работ вызывается по кнопке "Назначить сотр.". Рекомендуется использовать алгоритм "Назначать работы сотрудникам выравнивая нагрузку". Этот алгоритм балансирует число работ по видам работ и типам номеров, т.е. он пытается так распределить работы, чтобы у всех горничных для, например, выездной уборки было бы назначено примерно одинаковое количество люксов, двухместных номеров и т.д.
  4. После выполнения автоматического назначения сотрудников документ обычно редактируется вручную, т.е автоматическое назначение подправляется исходя из полученного распределения работ, этажей на которые попал тот или иной сотрудник и других условий.
  5. После того как всем работам назначен исполнитель документ может быть сохранен и распечатан. Печатать документ можно как в виде отдельных заданий на выполнение работ каждому сотруднику, так и спложным списком по номерам или сгруппированным по работам.
  6. В документе есть кнопка "Проверить", которую можно нажимать в течение дня и которая добавляет в документ новые работы, которые нужно выполнить в случае непредвиденных выселений гостей из номеров, завершения ремонтов и т.д. Измененные строки документа подсвечиваются жирным шрифтом. Выделение снимается, если зайти в режим редактирования строки и закончить редактирование клавишей Enter или комбинацией клавиш Shift+F2.

Относится к подсистеме: «Управление службой номеров».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Гостиница. Ссылается на справочник «Гостиницы».
  • Номер. Ссылается на справочник «Номерной фонд». Может выбираться группа номеров. Обычно используется в гостиницах, где, например, за каждый корпус отвечает свой супервайзер. В этом случае каждый супервайзер планирует работу по своему корпусу используя своих горничных.
  • Секция номеров. Ссылается на справочник «Секции номерного фонда». Аналогично группе номеров. Позволяет формировать документ не по всему номерному фонду а по его части.
  • Уборка по выезду. Ссылается на справочник «Работы». Параметр необходимо заполнить если планируется выполнять выездные уборки в номерах
  • Регулярная уборка. Ссылается на справочник «Работы». Параметр необходимо заполнить если планируется выполнять ежедневные уборки в занятых номерах 
  • Уборка в свободных номерах. Ссылается на справочник «Работы». Параметр необходимо заполнить если планируется выполнять уборки в свободных номерах. программа будет назначать номерам такие работы в случае, если номер простоял свободным более суток.
  • Уборка после окончания ремонта. Ссылается на справочник «Работы». Параметр необходимо заполнить если планируется выполнять уборки в номерах в которых закончился ремонт. Обычно заполняют выездной уборкой.
  • Набор регламентных работ. Ссылается на справочник «Наборы регламентных работ». Позволяет указать программе какие регламентные работы выполняют горничные в номерах. К регламентным работам обычно относят замену белья и замену полотенец. У каждой регламентной работы из набора регламентных работ может быть указана своя периодичность в днях.
  • Статус номера после выселения. Ссылается на справочник «Статусы номеров». Указывается тот статус номера, который программа устанавливает в номерах после выселения гостей. Обычно заполняется из настроек гостиницы.
  • Статус номеров после раннего (до 06:00) заезда. Ссылается на справочник «Статусы номеров». Если указан, то позволяет выполнять регулярную уборку в номерах, в которые был ночной заезд.
  • Статус занятых/грязных номеров. Ссылается на справочник «Статусы номеров». Указывается статус номеров, в которых нужно выполнять регулярную уборку. Этот статус обычно проставляется в занятых номерах специальной регламентной процедурой "Управление статусами занятых номеров" начиная со второго дня проживания гостя.
  • Статус номеров на инспекции. Ссылается на справочник «Статусы номеров». Специальный статус, если установлен у номера, то никакая работа ему назначена не будет
  • Отдел службы номеров. Ссылается на справочник «Отделы». Используется для отбора сотрудников в список сотрудников смены
  • Примечания. Тип значения: «Строка».
  • Автор. Ссылается на справочник «Сотрудники». Сотрудник создавший документ
  • Площадь номера. Тип значения: «Число». Общая площадь номеров, которым назначены работы по регистру нормативов работ
  • Продолжительность работ в часах. Тип значения: «Число». Общая продолжительность назначенных номерам работ по нормативам
  • Счетчик уборок по выезду. Тип значения: «Число». Поле используется для вывода итогового количества выездных уборок
  • Счетчик текущих уборок. Тип значения: «Число». Поле используется для вывода итогового количества регулярных уборок
  • Счетчик уборок после ремонтов. Тип значения: «Число». Поле используется для вывода итогового количества уборок в номерах после ремонтов
  • Счетчик уборок в свободных номерах. Тип значения: «Число». Поле используется для вывода итогового количества уборок в свободных номерах

Табличная часть «Работы»

  • Номер. Ссылается на справочник «Номерной фонд».
  • Тип номера. Ссылается на справочник «Типы номеров».
  • Статус номера. Ссылается на справочник «Статусы номеров». Статус номера на момент заполнения документа.
  • Время изменения статуса номера. Тип значения: «Дата». Время последнего на момент заполнения документа изменения статуса номера
  • Автор изменения статуса номера. Ссылается на справочники «Отделы», «Сотрудники». Автор последнего на момент заполнения документа изменения статуса номера
  • Вид блокировки номера . Ссылается на справочник «Виды блокировки номеров». Вид блокировки номера если номер на момент заполнения документа заблокирован (на ремонте, в аренде и проч.)
  • На брони. Тип значения: «Булево». Флажок включен, если на дату документа в номере планируется заезд по резервированию
  • Работа. Ссылается на справочник «Работы». Работа, которая должна быть выполнена в этом номере
  • Описание работы. Тип значения: «Строка». Строка с описанием состояния номера. Заполняется программой автоматически, не редактируется
  • Количество гостей в номере. Тип значения: «Число». Справочно количество гостей поживающих в номере
  • Гость. Ссылается на справочник «Клиенты (гости)». Гость проживающий в номере
  • Тип клиента. Ссылается на справочник «Типы клиентов». Тип проживающего или заезжающего в номер гостя. Используется для того, чтобы показать в каких номерах проживают, например, VIP гости
  • Гражданство. Ссылается на справочник «Страны». Страна гражданства проживающего гостя
  • Дата заезда. Тип значения: «Дата».
  • Дата выезда. Тип значения: «Дата».
  • День проживания. Тип значения: «Число». Номер дня проживания гостя. Используется для контроля назначения регламентных работ
  • Контрагент. Тип значения: «Строка», Ссылается на справочник «Контрагенты». Контрагент от которого проживает гость. Не редактируется, используется справочно
  • Документ основание. Ссылается на документы «Бронь», «Размещение», «Установить блокировку».
  • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник который назначен для выполнения работы
  • Продолжительность работы в минутах. Тип значения: «Число». Продолжительность выполненияработы по нормативам
  • Площадь номера. Тип значения: «Число». Площадь номера по нормативам выполнения работы
  • Счетчик уборок по выезду. Тип значения: «Число». Колонка используется для подсчета общего числа выездных уборок
  • Счетчик текущих уборок. Тип значения: «Число». Колонка используется для подсчета общего числа регулярных уборок
  • Счетчик уборок после ремонтов. Тип значения: «Число». Колонка используется для подсчета общего числа уборок после ремонтов
  • Счетчик уборок в свободных номерах. Тип значения: «Число». Колонка используется для подсчета общего числа уборок в свободных номерах
  • Это новая работа. Тип значения: «Булево». Скрытый флажок, который устанавливается кнопкой "Проверить" у новых работ
  • Это работа добавленная вручную. Тип значения: «Булево». Скрытый флажок, который устанавливается у работ добавленных в список работ вручную
  • Порядок сортировки гостиниц. Тип значения: «Число». Скрытый на форме реквизит. Порядок сортировки из карточки гостиницы
  • Порядок сортировки номеров. Тип значения: «Число». Скрытый на форме реквизит. Порядок сортировки из карточки номера
  • Порядок сортировки типов номеров. Тип значения: «Число». Скрытый на форме реквизит. Порядок сортировки из карточки типа номера
  • Порядок сортировки работы. Тип значения: «Число». Скрытый на форме реквизит. Порядок сортировки из карточки работы
  • Порядок сортировки сотрудников. Тип значения: «Число». Скрытый на форме реквизит. Порядок сортировки из карточки сотрудника

Табличная часть «Сотрудники»

  • Сотрудник. Ссылается на справочник «Сотрудники». Сотрудник в текущей смене
  • Отдел. Ссылается на справочник «Отделы». Отдел сотрудника
  • Группа номеров начиная с которой сотрудник выполнять работы не будет. Ссылается на справочник «Номерной фонд». Если заполнен позволяет указать, что этот сотрудник не выполняет работы например выше указанного этажа
  • Рабочих часов в день. Тип значения: «Число». Количество рабочих часов сотрудника в день. Заполняется вручную или автоматически из графика работы
  • Продолжительность работы в часах. Тип значения: «Число». Общая продолжительность назначенных сотруднику работ по нормативам
  • Площадь номера. Тип значения: «Число». Общая площадь назначенных сотруднику работ по нормативам
  • Счетчик уборок по выезду. Тип значения: «Число». Количество выездных уборок назначенных сотруднику
  • Счетчик текущих уборок. Тип значения: «Число». Количество регулярных уборок назначенных сотруднику
  • Счетчик уборок в свободных номерах. Тип значения: «Число». Количество уборок в свободных номерах назначенных сотруднику
  • Счетчик уборок после ремонтов. Тип значения: «Число». Количество уборок после ремонтов назначенных сотруднику
  • Порядок сортировки сотрудников. Тип значения: «Число». Скрытый реквизит. Порядок сортировки из карточки сотрудника. Сотрудники в смене по умолчанию показываются по возрастанию их порядка сортировки

Формы

  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.
  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Печатная форма работ по номерам.
  • Печатная форма работ по сотрудникам.
  • Печатная форма итогов по сотрудникам.

Способ вызова

Вызывается из главного меню программы по кнопке "Еще.../Закладка "2"/План работ"
