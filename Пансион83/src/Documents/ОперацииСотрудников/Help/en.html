Документ "Регистрация работы"

Назначение

Документ используется для регистрации в программе работ сотрудников и регистрации оборота номенклатуры в соответствии с нормативами расхода номенклатуры по видам работ.

Документ не вводится вручную. Документ создается автоматически в случае, когда используется программный интерфейс с АТС по регистрации работ горничных. Также документ автоматически формируется при вводе итогов выполненных сотрудниками работ через форму "Ввод итогов выполненных работ".

Относится к подсистеме: «Управление службой номеров».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Реквизиты

  • Сотрудник. Указывается сотрудник, выполнивший работу. При регистрации работы через АТС, сотрудник набирает свой персональный код, по которому и регистрируется работа. Персональный код сотрудника, набираемый на телефоне, указывается в справочнике "Сотрудники" в реквизите "Код доступа в АТС".
  • Гостиница. Ссылается на справочник «Гостиницы».
  • Тип номера. Ссылается на справочник «Типы номеров». Указывается тип номера, по которому была зарегистрирована работа.
  • Номер. Ссылается на справочник «Номерной фонд».  Номер комнаты, где выполнена работа.
  • Работа. Ссылается на справочник «Работы». Указывается вид работы. При использовании интерфейса АТС, код работы, набираемый на телефоне, должен быть указан у вида работы в реквизитах "Код начала работы" и "Код окончания работы".
  • Количество. Количество выполненных работ. При регистрации через АТС = 1. При создании через форму ввода итогов выполненных работ равно тому количеству которое было указано в итогах.
  • Время начала работы. Заполняется автоматически при получении команды с АТС.
  • Продолжительность работы в минутах. Вычисляется автоматически по данным времени начала и окончания работы. 
  • Время окончания работы. Заполняется автоматически при получении команды с АТС.
  • Площадь номера. Справочное значение, заполняется при выборе номера значением указанном у характеристики номера. Площадь номера задается в номере на закладке "Характеристики".
  • Количество человек. Количество гостей, проживающих в номере, и для которых выполняется работа. Используется когда расход номеклатуры зависит от количества гостей в номере.
  • Код начала работы для телефонной АТС. Полученный от АТС код начала выполнения работы.
  • Код окончания работы для телефонной АТС. Полученный от АТС код окончаниия выполнения работы.
  • Примечания. Комментарий к операции.
  • Автор. Ссылается на справочник «Сотрудники». Пользователь, создавший документ.

Табличная часть «Номенклатура»

Указывается номенклатура, которая была израсходована на выполнение работы. Заполняется по нормативам работы. Используется отчетом "Обороты по работам персонала" для расчета оборота расхода номенклатуры за выбранный период. По номенклатуре не ведется складской учет, т.е не считаеются остатки и проч. Данные обороты необходимы только для того, чтобы получить обороты, рассчитанные по нормативам и сравнить с фактическим расходом со склада.

  • Номенклатура. Ссылается на справочник «Номенклатура». Указывает какую номенклатуру израсходовали.
  • Количество номенклатуры на единицу измерения в штуках, граммах или миллилитрах.  Количество на одного человека или единицу площади.
  • Единица измерения. Тип значения: «Строка».
  • На каждого человека (гостя). Тип значения: «Булево». Если включено, то итоговое количество будет рассчитано, как произведение количества человек на количество за единицу.
  • На единицу площади номера. Тип значения: «Булево». Если включено, то итоговое количество будет рассчитано, как произведение площади на количество за единицу площади.
  • Количество номенклатуры в штуках, граммах или миллилитрах. Итоговое количество = Кол-во за единицу * кол-во человек (если норматив задан на человека) * площадь (если норматив задан на единицу площади).

Способ вызова

Вызывается из меню "Служба номеров/Регистрация работы сотрудника"

Движения

  • «Обороты по выполненным персоналом работам». Фиксируются обороты по количеству выполненных сотрудниками работ по видам работ и обороты по расходу номенклатуры для отчетности.
  • «Протокол выполнения работ персоналом». Фиксируется кто, когда и где выполнял работу. Позволяет отследить где находится горничная и какие работы она выполняла.
