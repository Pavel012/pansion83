Форма используется для массового быстрого ввода итогов по выполненным сотрудниками работам. Форму можно использовать в случаях, когда невозможно подключить автоматическую регистрацию работ сотрудников через АТС, но необходимо вести статистику по выполненным сотрудниками работам например для целей расчета заработной платы.
