Документ "Сторно"

Назначение

Документ используется для регистрации отмены начисления услуги по лицевому счету. Документ делает те же движения по регистрам, что и начисление, но с обратным знаком. В отличие от установки у начисления пометки на удаление, по операциям сторно можно формировать аудиторские отчеты. У каждой операции сторно определен автор и время проведения.

Относится к подсистеме: «Управление взаиморасчетами».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Начисление».

Реквизиты

  • Тип сторно. Ссылается на перечисление «Типы отмены действий». Выбирается пользователем из одного из предопределенных значений
  • Гостиница. Ссылается на справочник «Гостиницы». 
  • Документ основание. Ссылается на документы «Бронь», «Бронь ресурса», «Размещение», «Состав квоты». Заполняется автоматически из лицевого счета начисления
  • Начисление, по которому делается сторно. Ссылается на документ «Начисление». 
  • Примечания. Тип значения: «Строка». Обычно здесь указывается причина выполнения операции. Указывается автором операции.
  • Автор. Ссылается на справочник «Сотрудники». Автор операции.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка и выбора. Используется для отображения списка или выбора документов данного типа.

Способ вызова

Вызывается из фомы списка транзакций лицевого счета по кнопке "Сторно"
