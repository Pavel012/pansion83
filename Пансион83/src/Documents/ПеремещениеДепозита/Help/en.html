Документ "Перемещение депозита"

Назначение

Документ используется для регистрации перемещения суммы оплаты между лицевыми счетами. Например, в случае, когда по одному счету осталась предоплата и вместо оформления возврата эту предоплату нужно зачесть в счет оплаты услуг по другому лицевому счету.

В программе могут быть заданы ограничения на перемещения депозитов. В наборе прав пользователя можно запретить перемещение между лицевыми счетами разных групп гостей. Т.е. если гости поселены по разным заявкам, то между их лицевыми счетами нельзя выполнить перемещение депозита, а необходимо оформить возврат по одному лицевому счету и внести платеж по другому счету. Данное ограничение вводится в случаях, когда в бухгалтерии учет ведется в разрезе договоров и на каждую группу гостей в гостинице в бухгалтерии создается отдельный договор. Без этого ограничения в бухгалтерии не будет "сворачиваться" 62 счет в разрезе договоров.  

Относится к подсистеме: «Управление взаиморасчетами».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Фолио».

Реквизиты

  • Документ основание. Ссылается на документы «Бронь», «Бронь ресурса», «Размещение».
  • С лицевого счета. Ссылается на документ «Фолио». Указывается с какого счета переносится сумма. По этому счету будет уменьшена сумма оплаты.
  • На лицевой счет. Ссылается на документ «Фолио».  Указывается куда переносится сумма. По этому счету будет увеличена сумма оплаты.
  • Гостиница. Ссылается на справочник «Гостиницы».
  • Дата курса обмена. На эту дату рассчитывается курс.
  • Сумма в валюте лицевого счета источника. Указывается сумма, кторую нужно списать с лицевого счтета-источника.
  • Валюта лицевого счета источника. Ссылается на справочник «Валюты».  Заполняется автоматически данными фолио.
  • Курс обмена валюты лицевого счета источника по отношению к базовой валюте. Заполняется автоматически на дату курса. Используется для пересчета валюты.
  • Сумма в валюте лицевого счета получателя. Рассчитывается по курсу из суммы фолио-источника.
  • Валюта лицевого счета получателя. Ссылается на справочник «Валюты». Заполняется значением из счета-получателя.
  • Курс обмена валюты лицевого счета получателя по отношению к базовой валюте. Заполняется на дату курса.
  • Способ оплаты. Ссылается на справочник «Способы оплаты». Заполняется автоматически специальным предопределенным значением "Перенос депозита".
  • Секция оплаты. Ссылается на справочник «Секции оплаты». Заполняется автоматически из остатка фолио-источника, в случае, если учет в лицевых счетах ведется по секциям.
  • Примечания. Произвольный комментарий пользователя к операции.
  • Автор. Ссылается на справочник «Сотрудники». Пользователь выполнивший операцию.

Способ вызова

Открыть лицевой счет-источник и нажать на кнопку "Перенести депозит"
