Документ "Ввод начальных остатков по накопительным скидкам"

Назначение

Документ используется для регистрации начальных накопленных сумм ресурса в разрезе дисконтных карт/клиентов/контрагентов. Также документ можно использовать при необходимости задать начальную сумму бонусов на дисконтной карте или на карточке клиента. Документ позволяет внести начальные записи в регистр "Ресурсы накопительных скидок".

Относится к подсистеме: «Управление взаиморасчетами».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Непериодический

Тип номера: Строка

Реквизиты

  • Тип скидки. Ссылается на справочник «Типы скидок». Обязательное для заполнения поле, выбирается пользователем. Должен быть накопительным типом скидки.
  • Гостиница. Ссылается на справочник «Гостиницы». 
  • Примечания. Тип значения: «Строка». Обычно здесь указывается причина выполнения операции. Указывается автором операции.
  • Автор. Ссылается на справочник «Сотрудники». Автор операции.
  • Таблица измерений с значениями ресурсов или бонусов. Таблица в каждой строке которой для дисконтной карты/клиента/контрагента указываются начальная суммы ресурса (для обычных накопительных скидок) или сумма бонусов для типов скидок у которых указан коэффициент пересчета в бонусы.

Формы

  • Форма документа. Основная форма объекта. Используется для заполнения обязательных реквизитов.
  • Форма списка. Основная форма списка. Используется для отображения списка документов данного типа.

Способ вызова

Вызывается из фомы списка справочника "Типы скидок" по кнопке "Перейти" в панели инструментов списка или из меню "Взаиморасчеты/Скидки/Ввод начальных остатков по накопительным скидкам" полного интерфейса.
