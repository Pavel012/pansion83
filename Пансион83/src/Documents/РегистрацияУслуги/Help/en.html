Документ "Регистрация услуги"

Назначение

Документ регистрирует фактическое оказание услуги. Используется для учета фактически оказанных услуг и контроля оказания услуг. В отличие от документа "Начисление" не участвует в взаиморасчетах. Основная задача документа - показать, какими услугами клиенты фактически воспользовались из набора начисленных им услуг.

Например: при начислении по тарифу услуга завтрак будет запланирована к оказанию. Когда гость придет на завтрак ему зарегистрируют услугу и количество плановых завтраков уменьшится. Таим образом контролируется кто уже получил услугу (завтрак), а кто еще нет. Дополнительно получается отчет по фактически оказанному количеству завтраков на каждый день.

Относится к подсистеме: «Управление взаиморасчетами».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Фолио».

Реквизиты

  • Учетная дата. Дата фактического оказания услуги. 
  • Документ основание. Ссылается на документы «Бронь», «Бронь ресурса», «Размещение», «Состав квоты».
  • Гостиница. Ссылается на справочник «Гостиницы».
  • Группа гостей. Ссылается на справочник «Группы гостей».
  • Номер. Ссылается на справочник «Номерной фонд». Номер, в котором проживает гость.
  • Ресурс. Ссылается на справочник «Ресурсы». Указывается ресурс (конференц-зал и проч.), по которому начислена услуга.
  • Клиент (гость). Ссылается на справочник «Клиенты (гости)». Гость, которому начислена услуга.
  • Тип клиента. Ссылается на справочник «Типы клиентов». Свойство клиента.
  • Валюта лицевого счета. Ссылается на справочник «Валюты».
  • Лицевой счет. Ссылается на документ «Фолио». Лицевой счет, по которому была начислена услуга.
  • Услуга. Ссылается на справочник «Услуги». Услуга, оказание которой запланировано начислением и регистрируется данным документом.
  • Цена. Цена за единицу услуги.
  • Единица измерения. 
  • Количество. Количество регистрации услуги.
  • Сумма в валюте лицевого счета. Сумма регистрации услуги. 
  • Примечания. Комментарий к операции.
  • Автор. Ссылается на справочник «Сотрудники». Пользователь оформивший операцию. 

Формы

  • Форма остатков и оборотов. Используется для отражения гостей с запланированной услугой. 
  • Контроль услуги. Используется для быстрого поиска гостя и регистрации услуги на выбранного гостя. Поиск осуществляется по номеру комнаты или по карте гостя.
