Документ "Распределение аванса"

Назначение

Документ используется для распределения предоплаты юридического лица на его счета-требования по новым бронированиям или на группы гостей. Документ вводится в тех случаях, когда у контрагента с одной стороны есть аванс внесенный без указания номеров групп, а с другой стороны есть брони, по которым выставлены счета-требования. Документ, при проведении, перераспределяет сумму аванса контрагента таким образом, что сумма нераспределенного аванса уменьшается и аванс переносится на указанные группы гостей. Документ не изменяет общий баланс контрагента, но перераспределяет суммы авансов между группами гостей. Таблица документа может заполняться как по счетам-требованиям, так и по группам. Внешний вид таблицы задается типом операции: распределение аванса по счетам-требованиям или по группам.

Относится к подсистеме: «Управление взаиморасчетами».

Параметры

Длина номера: 12, тип номера: Строка

Периодичность номера: Год

Тип номера: Строка

Вводится на основании: «Контрагенты», «Счет-требование», «Договоры».

Реквизиты

  • Гостиница. Ссылается на справочник «Гостиницы».
  • Фирма. Ссылается на справочник «Фирмы».
  • Контрагент в бухгалтерии. Ссылается на справочник «Контрагенты».
  • Договор в бухгалтерии. Ссылается на справочник «Договоры».
  • Валюта взаиморасчетов. Ссылается на справочник «Валюты».
  • Группа гостей. Ссылается на справочник «Группы гостей». Группа, с которой "забирается" сумма аванса. Обычно пустая, т.к. перераспределяется в большинстве случаев не "привязанный" к группам аванс.
  • Сумма аванса. Тип значения: «Число». Сумма аванса по выбранной группе до перераспределения
  • Примечания. Тип значения: «Строка».
  • Автор. Ссылается на справочник «Сотрудники».
  • Вид операции. Ссылается на перечисление "Виды распределения аванса контрагента". Принимает значение: "по счетам требованиям" или "по группам гостей".

Табличная часть «Группы гостей»

  • Счет-требование. Ссылается на документ «Счет-требование». Колонка доступна когда вид операции установлен в "Распределение аванса по счетам требованиям".
  • Группа гостей. Ссылается на справочник «Группы гостей». Колонка доступна когда вид операции установлен в "Распределение аванса по группам".
  • Баланс по группе/счету-требованию. Тип значения: «Число». Вспомогательный реквизит, не редактируется. Показывает сумму, которая на момент создания документа еще должна быть оплачена по выбранному счету-требованию или выбранной группе.
  • Сумма в валюте взаиморасчетов с контрагентом. Тип значения: «Число». Сумма аванса, который пойдет на оплату баланса по счету-требованию или группе

Формы

  • Форма документа "Распределение аванса". Основная форма объекта. Используется для заполнения обязательных реквизитов.

Способ вызова

Вызывается из журнала "Счета-требования" по кнопке "Распределение аванса". Вызывается из журнала "Журнал взаиморасчетов с контрагентами" по кнопке "Распределение аванса".

Журналы отображающие документ: «Журнал взаиморасчетов с контрагентами».

Движения

  • «Взаиморасчеты по счетам».
  • «Взаиморасчеты с контрагентами».
