Отчет "Загрузка номеров"

Назначение

Имя отчета в конфигураторе «RoomInventory».

Отчет используется для контроля остатка свободных номеров на момент времени.

Вызывается автоматически в качестве расшифровки отчета с остаткам свободных номеров/мест по дням (Отчет "Загрузка по дням"). Показывает список брони, размещений, квот номеров, блокировок, которые действуют на указанный момент времени. Первая строка в отчете всегда показывает общее кол-во номеров в гостинице или по выбранной категории номеров. Остальные строки всегда идут со знаком минус, т.е. списывают номера из числа доступных к продаже. В итогах отчета показывается число номеров/мест оставшихся свободными. Списывать свободные номера могут: блокировки, квоты, бронь, размещения.

Отборы на форме отчета

  • По типу номера - задает отбор по типу номера или группе типов номеров.
  • По квоте номеров - задает отбор данных по квоте или группе квот. В строке с названием квоты и пустыми остальными полями в колонках "Номеров" и "Мест" показывается число номеров и мест добавленных в квоту.
  • По номеру - задает отбор по группе номеров или одному номеру комнаты. Если поле оставить пустым, то будут выведены данные по всем номерам гостиницы.

Группировки

  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает на организацию, юр. лицо, от имени которого было произведено бронирование (размещение).
  • Гость – гость, проживающий в номере, ссылка на справочник «Клиенты».
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Это размещение – признак того, что в текущей строке выводится размещение.
  • Это блокировка – признак того, что в текущей строке выводится количество заблокированных номеров по какому-либо виду блокировки. Двойной щелчок мышью по ячейке с названием вида блокировки откроет детальный отчет по заблокированным номерам.
  • Это бронь – признак того, что в текущей строке выводится бронь номера.
  • В наличии – булев флаг, истина только у первой строки отчета, которая всегда содержит общее количество номеров в гостинице или количество номеров в выбранной категории.
  • Это квота – признак того, что запись отчета это остаток номеров в квоте. Двойной щелчок мышью по ячейке с названием квоты откроет детальный отчет по квотам. 
  • Регистратор – ссылка на регистрирующий строку документ.
  • Номер – указан номер комнаты.
  • Квота номеров – указывает квоту номеров, ссылка на справочник «Квоты номеров».
  • Статус – ссылка на справочник «Статусы номеров», указывает статус номера, который обычно меняется при заезде и выселении из номера и обычно принимает значения «Занят», «Свободен», «Грязный», «Чистый».

Колонки

  • Вид размещения – ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Мест – число мест в данном номере.
  • Дата заезда – дата заезда по текущей брони (размещению).
  • Дата выезда – дата выезда по текущей брони (размещению).
  • Продолжительность – число дней, на которые забронирован номер (размещен гость).
  • Примечания – текстовый комментарий.
  • Номеров – число номеров.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).
