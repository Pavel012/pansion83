Отчет "Планируемые группы"

Назначение

Имя отчета в конфигураторе «ExpectedGuestGroups».

Отчет выводит таблицу планируемых загрузки номерного фонда по дням по предварительным группам, т.е. по брони в статусах с включенным флажком "Предварительная". На каждый день из периода отчета выводится забронированное кол-во номеров/мест в разрезе групп и типов номеров.

По каждой группе также рассчитывается средняя, минимальная и максимальные цены. 

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По квоте – по брони из выбранной квоты номеров
  • По типу номера - задает отбор по типу номера. Если поле оставить пустым, то будут выведены данные по всем типам номеров.
  • По контрагенту - задает отбор по контрагенту из карточек групп. Если поле оставить пустым, то будут выведены данные по всем контрагентам.
  • По группе - задает отбор по группе гостей. Если поле оставить пустым, то будут выведены данные по всем группам.
  • По статусу брони - задает отбор по статусу предварительной брони. Если поле оставить пустым, то будут выведены данные по всей брони со статусом с включенным флажком "Предварительная". 

Группировки

  • Контрагент – юр.лицо, заключившее с гостиницей договор, на основании которого произведено размещение гостя.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Тип номера – тип номера из брони.
  • Статус брони – Статус резервирования из брони.
  • Квота – Квота номеров из брони.
  • Дата – дата из периода отчета.

Колонки

  • Номеров – количество номеров, планируемых к заселению в данной группе.
  • Мест – количество мест, планируемых к заселению в данной группе.
  • Доп. мест – количество дополнительных мест, планируемых к заселению в данной группе.
  • Гостей – количество гостей, планируемых к заселению в данной группе.
  • Средняя цена группы – средняя стоимость проживания по группе в сутки.
  • Максимальная цена группы – максимальная стоимость проживания по группе в сутки.
  • Минимальная цена группы – минимальная стоимость проживания по группе в сутки.
