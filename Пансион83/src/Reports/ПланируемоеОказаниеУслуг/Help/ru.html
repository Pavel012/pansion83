Отчет "Планируемое оказание услуг"

Назначение

Имя отчета в конфигураторе «ServicesSalesForecast».

Отчет выводит планируемое оказание услуг на указанный период по активной брони и размещенным гостям. Обычно формируется на период в будущем.

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По группе - задает отбор по группе гостей, если поле оставить пустым, то будут выведены данные по всем группам гостей.
  • По номеру гостиницы.

Группировки

  • Учетная дата - дата оказания услуги.
  • День учета - день оказания услуги.
  • Час учета - час оказания услуги.
  • Месяц учета - номер месяца оказания услуги.
  • Квартал учета - номер квартала оказания услуги.
  • Неделя учета - номер недели оказания услуги.
  • Год учета - год оказания услуги.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Автор – пользователь, создавший документ «Бронь ресурса».
  • День заезда - день заезда для текущей группы.
  • Час заезда - час заезда для текущей группы.
  • Месяц заезда - месяц заезда для текущей группы.
  • Квартал заезда - квартал заезда для текущей группы.
  • Неделя заезда - неделя заезда для текущей группы.
  • Год заезда - год заезда для текущей группы.
  • Клиент - гость, ссылка на справочник «Клиенты».
  • Возраст гостя.
  • Гражданство гостя.
  • Город гостя.
  • Регион гостя.
  • Тип клиента – характеристика гостя, может использоваться как разделитель учета для аналитики, также используется для задания разных цен по разным типам клиентов (каждому типу клиента можно поставить в соответствие свой тариф).
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает на организацию, юр. лицо, от имени которого было произведено бронирование (размещение).
  • Фолио – лицевой счет.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой-либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета. Сторно.
  • Сторно - принимает значение «Да», если по данному размещению вводилось сторнирование услуги. В противном случае - значение «Нет».
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах. Количество дополнительных мест Количество мест Количество мест в номере Количество человек в номере Количество номеров
  • Количество дополнительных мест - количество дополнительных мест в брони или в размещении.
  • Количество мест - атрибуты документа на вкладке «Параметры».
  • Количество мест в номере - количество мест в номере согласно типу номера.
  • Количество человек - атрибуты документа на вкладке «Параметры».
  • Количество человек в номере - количество человек в номере согласно типу номера.
  • Количество номеров - атрибуты документа на вкладке «Параметры».
  • Документ основание – документ, на основании которого проводилось начисление. Ссылка на документ «Бронь», «Бронь ресурса», «Состав квоты», «Размещение».
  • Период - дата оказания услуги.
  • Цена - цена услуги с учетом всех комиссий и скидок.
  • Регистратор - документ, который начисляет услугу. Ссылка на документ «Бронь» или «Начисление».
  • Отчетная валюта – валюта, указанная в качестве отчетной для гостиницы.
  • Номер – указание номера комнаты в текущем размещении (брони). Тариф Тип тарифа
  • Тариф - показывает, по какому тарифу поселили гостя или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип тарифа - разделитель учета, в разрезе типа тарифа можно строить аналитические отчеты.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).
  • Услуга – ссылка на справочник «Услуги».
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов. Сумма Сумма по проживающим гостям Сумма по брони Сумма без НДС Сумма по проживающим гостям без НДС Сумма по брони без НДС Цель поездки
  • Цель поездки - ссылка на справочник «Цели визита», может использоваться для построения аналитических отчетов.
  • Ставка НДС – размер ставки НДС.

Колонки

  • Дата заезда – указание даты заезда для текущей группы.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС - та же сумма без НДС.
  • Комиссия по проживанию – сумма вычисленной комиссии для агента только для услуги «Проживание».
  • Комиссия без НДС по проживанию - та же сумма без НДС.
  • Комиссия по брони - сумма вычисленной комиссии для агента, коотрая указывается в документе «Бронь».
  • Комиссия без НДС по брони - та же сумма без НДС.
  • Сумма скидки - сумма скидки по проживанию и скидки по брони.
  • Сумма скидки без НДС - та же сумма без НДС.
  • Сумма скидки по проживанию - скидка за услугу «Проживание».
  • Сумма скидки без НДС по проживанию - та же сумма без НДС.
  • Сумма скидки по брони - сумма скидки (скидок) услуги (услуг) в документе «Бронь».
  • Сумма скидки без НДС по брони - та же сумма без НДС.
  • Количество - сумма значений Количество по проживающим гостям и Количество по бронию
  • Количество по проживающим гостям - количество услуг, начисленных фактически, согласно документу «Начисление».
  • Количество по брони - количество услуг, начисленных по брони.
  • Отклонение тарифа - разница между фактической стоимостью услуги и суммой по тарифу.
  • Сумма по тарифу - стоимость услуги по тарифу.
  • Примечания – текстовый комментарий.
  • Сумма - сумма значений «Сумма по проживающим гостям» и «Сумма по брони».
  • Сумма без НДС - та же сумма без НДС.
  • Сумма по проживающим гостям - сумма начисленная по услуге с учетом скидок и комиссий по размещению.
  • Сумма по проживающим гостям без НДС - та же сумма без НДС.
  • Сумма по брони - сумма начисленная по услуге с учетом скидок и комиссий по брони.
  • Сумма по брони без НДС - та же сумма без НДС.
