Отчет "Обороты по продажам типов номеров"

Назначение

Имя отчета в конфигураторе «RoomTypesSalesTurnovers».

Отчет позволяет получить итоги по продажам номеров в разрезе типов номеров. По каждому типу номера/группе типов номеров рассчитываются проценты продаж. Если указан период отчета в будущем, то отчет учитывает планируемые продажи по брони.

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По услуге - задает отбор по услуге, если поле оставить пустым, то будут выведенные данные по всем номерам гостиницы.
  • По номеру.
  • По типу номера.

Данные в отчете всегда группируются по типам номеров.

Колонки

  • Средняя цена места в номере (ADBR) – по данному типу номера выводится усредненная по всем размещениям в данном периоде цена места в номере.
  • Средняя цена места в номере (ADBR) без НДС – та же сумма за вычетом НДС.
  • Продано дополнительных мест – количество дополнительных мест, проданных в номерах данного типа за заданный период.
  • Средняя цена номера (ADR) – по данному типу номера выводится усредненная по всем размещениям за заданный период цена номера.
  • Средняя цена номера (ADR) без НДС – та же сумма за вычетом НДС.
  • Средняя продолжительность проживания гостя (ALS) – для данного типа номера выводится среднее по всем гостям число дней проживания.
  • Доступно мест в типе номера – общее количество мест в номерах данного типа за вычетом заблокированных мест.
  • Заблокировано мест в типе номера – число заблокированных мест в номерах данного типа.
  • Мест в типе номера – общее количество мест в номерах данного типа.
  • Продано мест – количество мест, проданных в номерах данного типа.
  • % Загрузки мест – отношение количества занятых мест в номерах данного типа к кол-ву мест в номерах данного типа.
  • % Продаж мест – отношение количества занятых мест в номере данного типа к количеству доступных мест в номерах данного типа.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – сумма предоставленной скидки за вычетом НДС.
  • Человеко-дни – количество человеко-дней. Рассчитывается по каждому гостю независимо от стоимости проживания.
  • Заезд гостей – количесво гостей, занимающих (планирующих занять) номер данного типа.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Отчетная валюта.
  • Средний доход с доступных к продаже мест (Rev. PAB) – отношение общего дохода с доступных к продаже мест в номерах данного типа к количеству доступных к продаже мест в номерах данного типа.
  • Средний доход с доступных к продаже мест (Rev. PAB) без НДС – та же сумма без НДС.
  • Средний доход с клиента за ночь (Rev. PAC) – отношение общей суммы продаж за период к общему числу человеко-дней.
  • Средний доход с клиента за ночь (Rev. PAC) без НДС – та же сумма за вычетом НДС.
  • Средний доход с доступных к продаже номеров (Rev. PAR) – отношение общего дохода доступных к продаже номеров данного типа к количеству доступных данного типа.
  • Средний доход с доступных к продаже номеров (Rev. PAR) без НДС – та же сумма без НДС.
  • Доходы от продажи номеров – сумма, начисленная только за услуги проживание гостям, занимающим номера данного типа.
  • Доходы от продажи номеров без НДС – та же сумма без НДС.
  • Доступно номеров в типе номера – общее количество номеров данного типа за вычетом заблокированных номеров данного типа.
  • Заблокировано номеров в типе номера – число заблокированных номеров данного типа.
  • Номеров в типе номера – общее количество номеров данного типа.
  • Продано номеров – количество номеров данного типа, занятых гостями.
  • % Загрузки номеров – отношение количества занятых номеров данного типа к кол ичеству номеров данного типа.
  • % Продаж номеров – отношение количества занятых номеров данного типа к количеству доступных номеров данного типа.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).
  • Продажи – сумма, начисленная за услуги (проживание и остальное) гостям, занимающим номера данного типа.
  • Продажи без НДС – та же сумма без НДС.
  • Всего мест – общее количество мест в гостинице (не зависит от типа номера).
  • Всего доступно мест – общее количество мест в гостинице за вычетом мест в заблокированных номерах (не зависит от типа номера).
  • Всего заблокировано мест – общее число мест в заблокированных номерах.
  • Общий % загрузки мест – отношение количества занятых мест в номерах данного типа к общему количеству мест в гостинице.
  • Общий % продаж мест – отношение количества занятых мест в номерах данного типа к общему количеству доступных мест в гостинице.
  • Всего номеров – общее количество номеров в гостинице (не зависит от типа номера).
  • Всего доступно номеров – общее количество номеров в гостинице за вычетом заблокированных (не зависит от типа номера).
  • Всего заблокировано номеров – общее количество заблокированных номеров в гостинице.
  • Общий % загрузки номеров – отношение количества занятых номеров данного типа к общему количеству номеров в гостинице.
  • Общий % продаж номеров – отношение количества занятых номеров данного типа к общему количеству доступных номеров в гостинице.
