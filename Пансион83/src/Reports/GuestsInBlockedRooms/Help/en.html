Отчет "Гости в заблокированных номерах"

Назначение

Имя отчета в конфигураторе «GuestsInBlockedRooms».

Отчет выводит список гостей проживавших в заблокированных номерах за указанный период по указанным гостиницам, номерам и типам номеров.

Отборы на форме отчета

  • По типу блокировки - задает отбор по типу блокировки в гостинице.
  • По типу номера - задает отбор по типу номера в гостинице.
  • По номеру - задает отбор по номеру, если поле оставить пустым, то будут выведенные данные по всем номерам гостиницы.

Группировки

  • Размещение - ссылка на справочник «Размещения», указывает на текущее размещение.
  • Гость – ссылка на справочник «Клиентов», значение гостя на данный момент, которого поселили.
  • Автор – сотрудник, оформивший блокировку номера.
  • Учетная дата начала блокировки - дата начала действия блокировки без времени.
  • День начала блокировки - день начала действия блокировки.
  • Час начала блокировки - час начала действия блокировки.
  • Месяц начала блокировки - номер месяца начала действия блокировки.
  • Квартал начала блокировки - номер квартала начала действия блокировки.
  • Неделя начала блокировки - номер недели начала действия блокировки.
  • Год начала блокировки - год начала действия блокировки.
  • Учетная дата окончания блокировки - дата окончания действия блокировки без времени.
  • Гостиница – название гостиницы, в которой размещен гость на данный момент. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Регистратор - ссылка на документ, зарегистрировавший блокировку в гостинице.
  • Номер – номер комнаты, в который был поселен гость.
  • Тип блокировки - ссылка на документ «Вид блокировки номеров».
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).

Колонки

  • Дата заезда - дата заезда гостя.
  • Дата выезда - дата выезда гостя.
  • Мест – количество мест в номере, занятых гостем.
  • Начало действия блокировки - дата и время начала действия блокировки.
  • Окончание действия блокировки - дата и время окончания действия блокировки.
  • Продолжительность - продолжительность действия блокировки в днях.
  • Количество - количество заблокированных номеров. Необходимо для подсчета итогового количества.
  • Блокировка снята - пинимает значение «Да», если блокировка номера завершена.
  • Период - период начала действия (до секунд).
  • Момент времени – показывает точное время оформления блокировки номера в базе данных.
  • Примечания – срока, значение из документа «Установить блокировку».
  • Номеров – количество номеров занятых данным гостем.
