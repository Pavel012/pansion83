Отчет "Загрузка гостиницы по дням"

Назначение

Имя отчета в конфигураторе «HotelOccupationDaily».

Отчет показывает остатки свободных номеров/мест в разрезе типов номеров.

У отчета есть 3 режима работы. Режимы работы переключаются установкой значения у поля "Продолжительность проживания".

  1. Продолжительность = 0. В этом режиме отчет показывает остатки свободных номеров/мест по часам. По умолчанию данные по часам свернуты и показываются остатки свободных номеров/мест на 23:59 (на конец дня) или 12:59 каждого дня. Если раскрыть день ("+" плюсиком), то остатки свободных номеров покажутся по часам. В настройках наборов прав сотрудников можно указать время, на которое по умолчанию будет показываться остаток свободных номеров/мест. Обычно для портье оставляют значение по умолчанию, т.е. остатки показывают на 23:59:59, для службы бронирования обычно время выставляют в 12:59:59, т.е. по умолчанию отчет показывает остатки на дату и время заезда по расчетному часу (+ 1 час для учета времени уборки). Этот режим обычно используется, когда нужно понять сколько у вас есть номеров для бронирования "на ночь". В этом режиме не "развернув" дату "плюсиком" по часам вы не увидите остатков с учетом ранних заездов/поздних выездов.
  2. Продолжительность = 1. В этом режиме отчет показывает минимальные остатки свободных номеров/мест на гостиничные сутки с расчетного часа по расчетный час. Расчетный час берется у тарифа, указанного в качестве тарифа по умолчанию в карточке гостиницы. В этом режиме вы видите сколько в гостинице осталось номеров/мест свободных для поселения на полные гостиничные сутки.  Этот режим показывает остатки с учетом ранних заездов и поздних выездов. Данные, которые показывает отчет в этом режиме совпадают с остатками, которые выгружаются в channel менеджеры и на сайт отеля. Этот режим формирования отчета является основным. 
  3. Продолжительность > 1. В этом режиме отчет показывает сколько у вас есть полностью свободных номеров/мест (без переселений) для размещения гостей на указанное число суток. В этом режиме отчет формируется не по остаткам свободных номеров по типам, а по списку номеров и показывает данные, которые совпадают с результатом ручного анализа графической карты загрузки номеров. Отчет рекомендуется формировать не более чем на 5 - 7 дней, т.к. иначе время формирования отчета может быть значительным. 

Отчет может показывать остатки свободных мест. Выбор режима осуществляется в панели инструментов отчета справа от таблицы отчета.

Двойной щелчок на количестве свободных номеров откроет расшифровку отчета, которая покажет, каким образом получилось количество свободных номеров (т.е. были ли размещения, брони или квоты на искомый момент).

Если в таблице отчета выделить период в строке какого-либо типа номера, то можно быстро создать на этот период новую бронь. Для этого нужно нажать в панели инструментов отчета справа от таблицы кнопку с "зеленым контурным треугольником".

Для обновления отчета используйте кнопку "Обновить" в панели инструментов отчета справа от таблицы отчета.

ПРи анализе данных отчета необходимо понимать что в первых двух режимах остатки показываются с учетом возможных переселений и оптимизаций расположения брони в графической карте загрузки номеров. Т.е. если в отчете показываются остатки, а в графической карте вы видите, что свободных номеров на нужный период нет, то это означает, что вы можете:

  • "передвинуть" брони с назначенными номерами комнат, чтобы разместить бронь оптимально
  • запланировать переселение проживающих гостей из одного номера в другой для того, чтобы освободить номер для бронирования 

Группировки

  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).  

Колонки

  • Свободно номеров/мест - количество свободных номеров или мест на указанный в колонке момент времени.
