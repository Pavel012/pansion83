Отчет "Должники"

Назначение

Имя отчета в конфигураторе «Debtors».

Отчет выводит список фолио (лицевых счетов) с ненулевым балансом. Если дата отчета заполнена, то баланс рассчитывается на эту дату. Если дата отчета "пустая", то берется окончательный баланс по фолио.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета взаиморасчетов – по контрагенту, договору.
  • По номеру - задает отбор по номеру, если поле оставить пустым, то будут выведенные данные по всем номерам.
  • По услуге - если указан, то отбираются только те лицевые счета, где были начисления по указанной услуге или группе услуг.
  • По типу отбора должников - тип может быть одним из предложенных: все должники; по которым принято решение; по которым не принято решение относительно их долга.
  • По агентам.

Группировки

  • Фолио – ссылка на документ «Лицевой счет», по которому произошло начисление.
  • Агент – значение–ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Автор фолио – пользователь, создавший документ «Лицевой счет».
  • Гость – элемент справочника «Клиенты».
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – договор из фолио, по которому проведено начисление.
  • Контрагент – юр.лицо, заключившее договор с гостиницей, на основании которого произведено размещение гостя.
  • Валюта фолио – валюта, в которой оформлен документ «Лицевой счет».
  • Дата заезда – дата и время заезда данного клиента.
  • День заезда – номер дня заезда в месяце.
  • Час заезда – указание часа заезда.
  • Месяц заезда – номер месяца заезда.
  • Квартал заезда – номер квартала заезда.
  • Неделя заезда – номер недели заезда.
  • Год заезда – год заезда.
  • Дата выезда – дата и время выезда данного клиента.
  • Описание фолио – ссылка на одноименное поле в документе Лицевой счет.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой – либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Фолио закрыто – признак закрытия лицевого счета. Для текущего фолио устанавливается в значение «истина», если данный счет полностью оплачен клиентом.
  • Главный в группе – булев флаг, указывающий для нескольких Клиентов в одной группе того, который оплачивает лицевые счета.
  • Документ основание – документ «Размещение».
  • Способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществляется оплата текущего гостя.
  • Секция – секция ККМ, по которой происходит оплата (ссылка на одноименное поле документа Лицевой счет).
  • Номер – номер комнаты, в котором размещен Клиент.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Это долг клиента - принимает значение «Да», если это долг клиента и «Нет», если гость заехал от контрагента и контрагент является плательщиком.
  • Это долг контрагента - принимает значение «Да», если это долг контрагента и «Нет», если гость расплачивается за себя сам.

Колонки

  • Баланс – указывает на состояние взаиморасчетов клиента и гостиницы по лицевому счету. Баланс положителен если это долг гостя, и отрицателен в случае долга гостиницы. Баланс рассчитывается на дату отчета.
  • Баланс клиента – в данной колонке показывается долг клиента. Баланс рассчитывается на дату отчета. 
  • Баланс контрагента – если в лицевом счете плательщиком указан контрагент, то в данной колонке будет показан долг контрагента по лицевому счету клиента. Баланс рассчитывается на дату отчета.
  • Баланс на текущее время - баланс по лицевому счету, который при любых значениях даты отчета рассчитывается на текущие дату и время.
  • Лимит по преавторизациям - сумма преавторизации по кредитным картам. Баланс по преавторизациям рассчитывается на дату отчета. 
  • Превышение суммы оказанных услуг над суммой лимита по преавторизации - разница между балансом клиента и лимитом преавторизаций.
  • Сумма, планируемая к оказанию - оставшаяся сумма, на величину которой нужно оказать услуги до даты отчета. Рассчитывается как разность между балансом на дату отчета и балансом на текущие дату и время. 
  • Примечания – текстовый комментарий в лицевом счете.
