Отчет "История бронирования номеров"

Назначение

Имя отчета в конфигураторе «ReservationsHistory».

Отчет выводит всю бронь за указанный период по указанным гостиницам, номерам и типам номеров.

Отборы на форме отчета

  • По виду проверки периода отчета - проверка дат брони относительно периода формирования отчета. Проверок существует несколько видов:
    • пересечение - в отчете отобразятся брони, период проживания которых пересекается с периодом отчета;
    • начинается в периоде - в отчет попадут только те брони, дата заезда которых попадает в период отчета;
    • заканчивается в периоде - в отчет попадут только те брони, дата выезда которых попадает в период отчета;
    • начинается или заканчивается в периоде - в отчет попадут только те брони, дата заезда или выезда которых попадает в период отчета;
    • дата создания документа попадает в период - в отчете отобразятся брони, которые были созданы в периоде отчета.
  • По номеру - задает отбор по номеру, если поле оставить пустым, то будут выведены данные по всем номерам гостиницы.
  • По типу номера - задает отбор по типу номера, если поле оставить пустым, то будут выведены данные по всем типам номеров гостиницы.
  • По типу брони - может быть действующей (т.е. введена, имеет статус «Бронь», но по ней еще не заехали гости) и недействующей.

Группировки

  • Счет (Счет–требование или Фолио) – ссылка на счет, выставленный плательщику.
  • Валюта счета – валюта документа Счет–требование или Фолио.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Автор – пользователь, автор документа «Бронь».
  • День заезда - день заезда по текущей брони.
  • Час заезда - час заезда по текущей брони.
  • Месяц заезда - номер месяца
  • Квартал заезда - номер квартала заезда по брони.
  • Неделя заезда - номер недели
  • Год заезда - год заезда по брони.
  • Тип клиента – возможность указать бесплатного клиента.
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Контактное лицо – ссылка на одноименное текстовое поле в документе «Бронь», указывает на лицо, бронировавшее данный номер от имени клиента. Обычно указывается при бронировании для контрагентов.
  • Договор – основание для расчетов между юр.лицом гостиницы и контрагентом.
  • Валюта брони.
  • Контрагент – юр.лицо, заключившее с гостиницей договор, на основании которого произведено размещение гостя.
  • Тип контрагента – справочник «Типы контрагентов».
  • Дисконтная карта – ссылка на справочник дисконтных карт. Заполняется при поселении, если гость предъявил карту.
  • Тип скидки – разделитель учета, ссылка на справочник «Тип скидок». Установка соответствия между набором услуг и размером скидки.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Путевка/курсовка – ссылка на путевку (курсовку).
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Планируемый способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществляется оплата текущего гостя.
  • Статус брони – статус текущей брони.
  • Номер – номер комнаты для резервирования.
  • Квота номеров – элемент справочника «Квоты номеров».
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип тарифа – элемент справочника «Типы тарифов».
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором планируется размещение гостя.
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.
  • Цель поездки – ссылка на справочник «Цели визита», может использоваться для построения аналитических отчетов.

Колонки

  • Вид размещения – вид размещения, соответствующий текущей брони.
  • Баланс по счету – состояние взаиморасчетов с данным гостем.
  • Комиссия агента – сумма комиссии, начисленной агенту.
  • Тип комиссии агента.
  • Сумма брони - стоимость проживания и всех услуг по брони.
  • Причина аннуляции - причина аннуляции брони.
  • Автор аннуляции - пользователь, который провел аннуляцию.
  • Автомобиль – соответствующее поле документа «Бронь».
  • Дата заезда – дата заезда по текущей брони.
  • Дата выезда – дата выезда по текущей брони.
  • Количество оставшихся дней до заезда.
  • Скидка – размер скидки по отношению к тарифу.
  • Продолжительность – продолжительность предполагаемого заезда.
  • Гость – гость, справочник «Клиенты».
  • Возраст счета в днях - количество дней, прошедших после даты выставления Счета–требования (Фолио).
  • Главный в группе – булево значение флажка, отмечающего главного в группе, на которого будут произведены все начисления по группе.
  • Доп. мест – количество дополнительных мест, имеющихся в данном номере.
  • Мест – количество мест, имеющихся в данном номере.
  • Заезд доп. мест – количество дополнительных мест, зарезервированных в текущей группе.
  • Заезд мест – количество дополнительных мест, зарезервированных в текущей группе.
  • Заезд гостей – количество гостей, планируемых к размещению в данной группе.
  • Заезд номеров – количество номеров, зарезервированных в текущей группе.
  • Гостей – количество гостей, ожидаемых в данной группе.
  • Количество человек в номере - указывается возможное количество гостей в номере.
  • Номеров – количество номеров, резервируемых текущей бронью.
  • Документ основание - документ, на основании которого была создана бронь.
  • Регистратор – ссылка на документ «Бронь».
  • Примечания - текстовое пояснение.
  • Количество – количество номеров.
  • Сумма счета – сумма выставленного счета.
  • Оплата по счету – сумма оплаты по счету.
