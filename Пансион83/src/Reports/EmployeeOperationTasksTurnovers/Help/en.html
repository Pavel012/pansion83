Отчет "Обороты по заданиям на выполнение работ в номерах"

Назначение

Имя отчета в конфигураторе «EmployeeOperationTasksTurnovers».

Отчет формирует оценку количества работ, которые должны были быть выполнены персоналом гостиницы в номерах. Данные этого отчета можно сравнивать с данными отчета по фактически выполненным работам или сравнивать с фактическим расходом белья, моющих средств и т.д. со склада гостиницы.

Данный отчет задает плановые количества работ исходя из занятости номерного фонда и правил выполнения регламентных работ принятых в гостинице.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета выполнения работ – по работе, по набору регламентных работ.
  • По номеру - задает отбор по номеру. Если поле оставить пустым, то будут выведенные данные по всем номерам.
  • По наббору услуг и услугам - если поля оставить пустыми, то будут выведенные данные по всем типам услуг и по всем услугам.
  • По типу номера - задает отбор по типу номера. Если поле оставить пустым, то будут выведенные данные по всем номерам.

Группировки

  • Выездная уборка – наименование работы для выездной уборки. Может быть использовано вместо кода.
  • Гражданство гостя – ссылка на справочник стран, содержит страну гражданства гостя.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Текущая уборка в занятом номере.
  • Регулярная работа – наименование регулярной работы, может быть использовано вместо кода.
  • Уборка после ремонта - указывается вид работ, который нужно выполнить после окончания ремонта в номере. В день окончания ремонта в номере программа автоматически подставит этот вид уборки для номера.
  • Номер – номер комнаты. Выводится всегда, если в номере не нужно проводить никаких работ, то поля с кодами работ остаются пустые.
  • Тип блокировки – ссылка на справочник «Типы блокировки», содержит значение установленное документов блокировки.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).
  • Уборка в свободном номере - указывается вид работы, которую нужно выполнить, если в номере никто не живет второй день, и номер простаивает свободным.

Колонки

  • Дата заезда – дата заезда гостя в номер.
  • Код выездной уборки – краткий буквенный код для выездной уборки.
  • Количество выездных уборок.
  • Код сортировки выездной уборки работы – реквизит справочника работ, используется для сортировки в отчете.
  • Дата выезда – дата выезда гостя из номера. Если гость еще не выехал, то показывается планируемая дата выезда.
  • Гость – ссылка на справочник клиентов.
  • Код текущей уборки в занятых номерах – краткий буквенный код текущей уборки в занятом номере.
  • Количество текущих уборок в занятых номерах.
  • Код сортировки текущей уборки работы – реквизит справочника работ, используется для сортировки в отчете.
  • Код регулярной работы – краткий буквенный код периодической регламентной работы, например, смены белья.
  • Количество регулярных работ.
  • Код сортировки регулярной работы – реквизит справочника работ, используется для сортировки в отчете.
  • Код уборки в номерах после ремонта – краткий буквенный код уборки, которую нужно выполнить в номере в день окончания ремонта.
  • Количество уборок после ремонта.
  • Код сортировки уборки в номерах после ремонта – реквизит справочника работ, используется для сортировки в отчете.
  • Код уборки в свободных номерах - краткий буквенный код уборки номеров, которые простаивают свободными. Если в гостинице не принято убирать свободные номера, то колонку можно исключить из отчета.
  • Количество уборок в свободных номерах.
  • Код сортировки уборки в свободных номерах – реквизит справочника работ, используется для сортировки в отчете.
