Отчет "Обороты по работам персонала"

Назначение

Имя отчета в конфигураторе «EmployeeOperationsTurnovers».

Отчет используется для анализа работ горничных и расчета итогов по количеству выполненных работ по видам работ, итогов по фактической продолжительности уборок и по фактической площади убранных помещений, итогов по расходу номенклатуры.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета выполнения работ – по сотруднику, по работе, по номенклатуре.
  • По номеру - задает отбор по номеру. Если поле оставить пустым, то будут выведенные данные по всем номерам.
  • По типу номера - задает отбор по типу номера. Если поле оставить пустым, то будут выведенные данные по всем номерам.

Группировки

  • Учетная дата – дата регистрации работы в программе (без указания времени).
  • Учетный месяц – номер месяца регистрации работы в программе.
  • Учетный квартал – номер квартала регистрации работы в программе.
  • Учетная неделя – номер недели в году регистрации работы в программе.
  • Учетный год – год выполнения работы регистрации работы в программе.
  • Номенклатура – ссылка на справочник «Номенклатура», указывает на номенклатуру, использованную в работе.
  • Сотрудник – сотрудник, выполнявший работу.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Работа – выполненная работа.
  • Номер – указание номера комнаты.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).
  • Единица измерения количества номенклатуры.

Колонки

  • Количество работ – указание количества выполненной работы.
  • Оборот по продолжительности работ в часах – длительность работы в часах.
  • Оборот по продолжительности работ в минутах – длительность работы в минутах.
  • Оборот по количеству человек – количество человек, выполнявших работу.
  • Количество номенклатуры – количество номенклатуры, использованной при выполнении работы.
  • Оборот по площади номеров – площадь номера.
