Отчет "Обороты по продажам номеров"

Назначение

Имя отчета в конфигураторе «RoomSalesTurnovers».

Отчет позволяет получить итоги по продажам номеров в разрезе измерений регистра «Продажи номеров». Отчет можно использовать для анализа больших объемов данных. При формировании на будущие даты отчет учитывает прогноз продаж по брони.

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По услуге - задает отбор по услуге, если поле оставить пустым, то будут выведены данные по всем номерам гостиницы.
  • По номеру - обычно задает отбор по папке справочника "Номерной фонд", т.е. по корпусу или этажу. 
  • По типу номера.
  • Календарь - если заполнен, то позволяет в качестве группировки данных отчета использовать периоды (типы дней) выбранного календаря.
  • Отбор по периоду бронирования - если заполнен, то в отчет попадают только те продажи, по которым бронирование было выполнено в указаном периоде.
  • Учитывать "жесткие" блоки - если галочка включена, то отчет добавляет к числу проданных номеров/мест остаток свободных номеров в квотах с включенной галочкой <Это "жесткий" блок>. Если у таких квот указан тариф, то стоимость проживания в номере по этому тарифу умноженная на остаток номеров в квоте добавляется к суммам продаж. Стоимость проживания берется по виду размещения с типом "Весь номер" с наименьшим порядком сортировки.

Группировки

  • Вид размещения - ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Учетная дата – дата продажи услуги проживания (без указания времени).
  • Учетный месяц - номер месяца продажи услуги проживания.
  • Учетный квартал - номер квартала продажи услуги проживания.
  • Учетная неделя - номер недели продажи услуги проживания.
  • Номер дня недели - номер дня недели.
  • День недели - номер+название дня недели.
  • Учетный год - год продажи услуги проживания.
  • Тип дня календаря - ссылка на справочник «Типы дней календаря».
  • Тип клиента – характеристика гостя, может использоваться как разделитель учета для аналитики, также используется для задания разных цен по разным типам клиентов (каждому типу клиента можно поставить в соответствие свой тариф).
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Документ основание – Фолио или Размещение.
  • Отчетная валюта – валюта, указанная в качестве отчетной для данной гостиницы.
  • Номер – указывает номер в гостинице.
  • Цена проживания - стоимость проживания по тарифу.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).
  • Услуга – услуга, по которой прошло начисление.
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.

Колонки

  • Продано дополнительных мест – количество проданных дополнительных мест.
  • Средняя цена места - средняя цена места в номере.
  • Средняя цена места без НДС - та же сумма без НДС
  • Средняя цена номера - отношение дохода с данного номера на количество человеко-дней за отчетный период.
  • Средняя цена номера без НДС - та же сумма без НДС.
  • Ремонт мест - количество мест, находящихся в ремонте.
  • Заблокировано мест в типе номера - количество мест в номере, на который есть блокировка.
  • Всего мест - общее количество мест в гостинице.
  • Мест в типе номера - количество мест в контретном типе номера.
  • Продано мест – количество проданных мест.
  • % Загрузки мест - отношение проданых мест к числу общих мест в гостинице (Всего мест) * 100%.
  • % Загрузки мест по типу номера за день - отношение проданых мест данного типа за день к числу общих мест в номере данного типа * 100%.
  • % Продаж мест - отношение проданых мест к числу доступных к продаже мест в гостинице (за вычетом заблокированных номеров) * 100%.
  • % Продаж мест по типу номера за день - отношение проданых мест данного типа за день к числу доступных к продаже мест данного типа номера (за вычетом заблокированных номеров) * 100%.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма вычисленной комиссии для агента за вычетом НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – сумма предоставленной скидки за вычетом НДС.
  • Человеко-дни – количество человеко-дней. Рассчитывается по каждому гостю независимо от стоимости проживания.
  • Заезд гостей – количество гостей, заезжающих в номер.
  • Средний доход с места (RevPAB) - отношение общего дохода с доступных к продаже мест в номерах данного типа к кол-ву доступных к продаже мест номерах данного типа.
  • Средний доход с места (RevPAB) без НДС - та же сумма без НДС.
  • Средний доход с номера (RevPAR) - отношение общего дохода с доступных к продаже номеров данного типа к количеству доступных данного типа.
  • Средний доход с номера (RevPAR) без НДС - та же сумма без НДС.
  • Доходы от продажи номеров – сумма продаж только услуг проживания для номеров данного типа.
  • Доходы от продажи номеров без НДС – предыдущая сумма за вычетом суммы НДС.
  • Ремонт номеров - количество номеров, находящихся в ремонте.
  • Заблокировано номеров в типе номера - количество заблокированных номеров конкретного типа.
  • Всего номеров - общее количество номеров в гостинице.
  • Номеров в типе номера - количество номеров в гостинице данного типа.
  • Продано номеров – количество проданных номеров данного типа.
  • % Загрузки номеров - отношение проданых номеров к числу общих номеров в гостинице (Всего номеров) * 100%.
  • % Загрузки номеров по типу номера за день - отношение проданых номеров данного типа за день к числу номеров данного типа * 100%.
  • % Продаж номеров - отношение проданых номеров к числу доступных к продаже номеров в гостинице (за вычетом заблокированных номеров) * 100%.
  • % Продаж номеров по типу номера за день - отношение проданых номеров данного типа за день к числу доступных к продаже номеров данного типа номера (за вычетом заблокированных номеров) * 100%.
  • Продажи – сумма продаж всех услуг в номерах данного типа (проживание + дополнительные).
  • Продажи без НДС – предыдущая сумма за вычетом суммы НДС.
  • Общие продажи - итоговое значение всех продаж.
  • Общий % продаж - отношение вида Продажи/Общие продажи.
