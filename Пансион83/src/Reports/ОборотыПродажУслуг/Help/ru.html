Отчет "Обороты продаж услуг"

Назначение

Имя отчета в конфигураторе «ServiceSalesTurnovers».

Отчет позволяет получить итоги по продажам услуг в разрезе измерений регистра "Продажи услуг".

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По группе - задает отбор по группе гостей, если поле оставить пустым, то будут выведены данные по всем группам.
  • По контагенту.
  • По договору.

Группировки

  • Учетная дата – дата оказания услуги.
  • Учетный месяц – номер месяца оказания услуги.
  • Учетный квартал – номер квартала оказания услуги.
  • Учетная неделя – номер недели оказания услуги.
  • Учетный год – год оказания услуги.
  • Тип клиента – характеристика гостя, может использоваться как разделитель учета для аналитики, также используется для задания разных цен по разным типам клиентов (каждому типу клиента можно поставить в соответствие свой тариф).
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Фолио – ссылка на документ «Лицевой счет».
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Документ основание – ссылка на документ «Размещение».
  • Способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществляется оплата текущего гостя.
  • Отчетная валюта – валюта, указанная в качестве отчетной для данной гостиницы.
  • Услуга – название оказанной услуги.
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.

Колонки

  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – та же сумма без НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – та же сумма без НДС.
  • Цена – цена оказанной услуги.
  • Количество – количество оказанной услуги.
  • Сумма – сумма, на которую оказана услуга.
  • Сумма без НДС – та же сумма без НДС.
