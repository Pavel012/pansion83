Отчет "Планируемые продажи ресурсов"

Назначение

Имя отчета в конфигураторе «ResourceSalesForecast».

Отчет выводит планируемое оказание услуг на указанный период по активной брони ресурсов и по фактически начисленным по брони ресурсов услугам. Обычно формируется на период в будущем.

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По услуге - задает отбор по услуге, если поле оставить пустым, то будут выведены данные по всем услугам гостиницы.
  • По ресурсу.

Группировки

  • День учета - день оказания услуги.
  • Час учета - час оказания услуги.
  • Месяц учета - номер месяца оказания услуги.
  • Квартал учета - номер квартала оказания услуги.
  • Неделя учета - номер недели оказания услуги.
  • Год учета - год оказания услуги.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Автор – пользователь, создавший документ «Бронь ресурса».
  • Клиент – гость, ссылка на справочник «Клиенты».
  • Возраст гостя.
  • Гражданство гостя.
  • Город гостя.
  • Регион гостя.
  • Тип клиента – характеристика гостя, может использоваться как разделитель учета для аналитики, также используется для задания разных цен по разным типам клиентов (каждому типу клиента можно поставить в соответствие свой тариф).
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает на организацию, юр. лицо, от имени которого было произведено бронирование (размещение).
  • Фолио – лицевой счет.
  • День начала - день начала действия брони ресурса.
  • Час начала - час начала действия брони ресурса.
  • Месяц начала - месяц начала действия брони ресурса.
  • Квартал начала - квартал начала действия брони ресурса.
  • Неделя начала - неделя начала действия брони ресурса.
  • Год начала - год начала действия брони.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Сторно.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Количество человек – число пользователей ресурса из текущей группы гостей.
  • Документ основание - ссылка на документ «Бронь ресурса».
  • Период.
  • Цена – указывает цену ресурса.
  • Регистратор – ссылка на документ «Начисление».
  • Отчетная валюта – валюта, указанная в качестве отчетной для гостиницы.
  • Ресурс – ссылка на справочник ресурсов, указывает на проданный ресурс.
  • Тип ресурса – указывает на тип забронированного ресурса.
  • Услуга – ссылка на справочник «Услуги».
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.
  • Ставка НДС – размер ставки НДС.

Колонки

  • Учетная дата - дата оказания услуги.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Фактически начисленная сумма комиссии – фактическая сумма вычисленной комиссии для агента.
  • Планируемая сумма комиссии – планируемая сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма вычисленной комиссии для агента за вычетом суммы НДС.
  • Фактически начисленная сумма комиссии без НДС – фактическая сумма вычисленной комиссии для агента за вычетом суммы НДС.
  • Планируемая сумма комиссии без НДС– планируемая сумма вычисленной комиссии для агента за вычетом суммы НДС.
  • Время начала брони.
  • Время окончания брони.
  • Сумма скидки – сумма предоставленной скидки.
  • Фактически начисленная сумма скидки – сумма предоставленной скидки по факту.
  • Планируемая сумма скидки – сумма предоставленной скидки по плану.
  • Сумма скидки без НДС– сумма предоставленной скидки за вычетом суммы НДС.
  • Фактически начисленная сумма скидки без НДС – сумма предоставленной скидки по факту за вычетом суммы НДС.
  • Планируемая сумма скидки без НДС – сумма предоставленной скидки по плану за вычетом суммы НДС.
  • Продолжительность – количество забронированных часов ресурса.
  • Часов работы ресурсов – длительность работы ресурса в сутки (обычно в часах). Например, если ресурс доступен круглосуточно, то значение будет 24.
  • Фактически продано часов работы ресурсов – продолжительность работы ресурса по факту начисления услуги.
  • Планируемое число часов работы ресурсов по брони – продолжительность работы ресурса по факту бронирования ресурса.
  • Количество – количество проданного ресурса.
  • Фактически оказанное количество – количество проданного ресурса по факту оказания услуги.
  • Планируемое количество по брони – количество проданного ресурса по факту бронирования ресурса.
  • Доходы от продажи ресурсов.
  • Фактически начисленные доходы от продажи ресурсов.
  • Планируемые доходы от продажи ресурсов по брони.
  • Доходы от продажи ресурсов без НДС.
  • Фактически начисленные доходы от продажи ресурсов – доходы от продажи ресурса по факту.
  • Планируемые доходы от продажи ресурсов по брони без НДС – доходы от продажи ресурса по плану.
  • Сумма – сумма продажи ресурса.
  • Фактическая сумма продаж – фактическая сумма продажи ресурса.
  • Планируемая сумма продаж – планируемая сумма продажи ресурса.
  • Сумма без НДС – сумма продажи ресурса за вычетом суммы НДС.
  • Фактическая сумма продаж без НДС – фактическая сумма продажи ресурса за вычетом суммы НДС.
  • Планируемая сумма продаж по брони без НДС – планируемая сумма продажи ресурса за вычетом суммы НДС.
