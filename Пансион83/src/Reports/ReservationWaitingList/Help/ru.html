Отчет "Лист ожидания брони"

Назначение

Имя отчета в конфигураторе «ReservationWaitingList».

Отчет позволяет распечатать список не действующей брони, которая находится в листе ожидания.

Отборы на форме отчета

  • По флагу возможности подтверждения брони - показывать весь лист ожидания или только ту бронь в листе ожидания, которую можно разместить.
  • По гостинице - задает отбор по отелю. Если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По типу номера - задает отбор по типу номера, если поле оставить пустым, то будут выведены данные по всем типам номеров гостиницы.

Группировки

  • Рейтинг брони - задается вручную числом для сортировки в отчете.
  • Бронь может быть подтверждена – обработка «Проверить лист ожидания брони» анализирует состояние брони и показывает возможно ли подтвердить бронь на данный момент времени.
  • Бронь – ссылка на документ «Бронь».
  • Дата заезда – дата заезда по текущей брони.
  • Дата выезда – дата выезда по текущей брони.
  • Группа гостей – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором планируется размещение.

Колонки

  • Бронь дополнительных мест – число дополнительных мест по данным текущей брони.
  • Бронь мест – число мест по данным текущей брони.
  • Бронь гостей – число гостей по данным текущей брони.
  • Бронь номеров – число номеров по данным текущей брони.
  • Количество резервирований.
