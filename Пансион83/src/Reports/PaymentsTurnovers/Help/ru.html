Отчет "Обороты по платежам"

Назначение

Имя отчета в конфигураторе «PaymentsTurnovers».

Отчет позволяет получить итоги по платежам в разрезе измерений регистра "Обороты по платежам".

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета взаиморасчетов - по контрагенту, по договору.
  • По группе - задает отбор по группе гостей.
  • По фирме - задает отбор по фирме, если поле оставить пустым, то будут выведены данные по всем фирмам.

Группировки

  • Учетная дата – дата платежа (без указания времени).
  • Учетный месяц – номер месяца платежа.
  • Учетный квартал – номер квартала.
  • Учетная неделя – номер недели платежа.
  • Учетный год – год платежа.
  • Автор – пользователь, создавший документ «Платеж».
  • ККМ – указывается ККМ, через которую прошел платеж (в случае наличной оплаты).
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Документ основание – размещение или фолио.
  • Плательщик – гость или контрагент.
  • Валюта платежа – валюта документа «Платеж».
  • Способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществляется оплата текущего гостя.
  • Кассовая секция – указание номера кассовой секции, по которой прошел платеж.

Колонки

  • Период – дата и время до секунд.
  • Платеж – ссылка на документ «Платеж»(наличная оплата) или «Акт» (безналичная оплата).
  • Сумма – общая сумма, которая равна сумма платежей за вычетом суммы возвратов.
  • Сумма возвратов – по данному клиенту показана сумма возвратов.
  • Сумма платежей – по данному клиенту показана сумма платежей.
  • Ставка НДС – ссылка на справочник «Ставки НДС».
  • Сумма НДС – НДС для колонки «Сумма».
  • Сумма НДС возвратов – НДС для колонки «Сумма возвратов».
  • Сумма НДС платежей – НДС для колонки «Сумма платежей».
