Отчет "Прогноз продаж по контрагентам"

Назначение

Имя отчета в конфигураторе «CustomerSalesForecast».

Отчет выводит планируемое оказание услуг на указанный период по активной брони и размещенным гостям. Обычно формируется на период в будущем. Отчет формируется по услугам оплачиваемым контрагентами, если услуги оплачиваются самим гостем, то они группируются под "пустым" контрагентом.

Отборы на форме отчета

  • Период отчета – задает период, в который были начисления.
  • Гостиница – задает отбор по гостинице, если поле оставить пустым, то будут выведенные данные по все гостиницам.
  • Номер – задает отбор по номеру комнаты или по группе номеров.
  • Тип номера – задает отбор по типу номера или по группе типов номеров.
  • Основные параметры взаиморасчетов – по контрагенту, по договору, по группе.
  • Набор услуг – если поле заполнено, то в отчет попадут только данные по услугам входящим в выбранный набор.

Группировки

  • Вид размещения — ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Учетная дата – дата учета услуги программой.
  • День учета - число месяца учета услуги.
  • Час учета - час учета услуги.
  • Месяц учета - номер месяца учета услуги.
  • Квартал учета - номер квартала.
  • Неделя учета - номер недели в году.
  • Год учета - год.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Тип комиссии - значение типа комиссии, может быть одно из значений перечисления «Виды комиссий агентам». Стандартное значение – «%», процент.
  • Автор - автор документа, по которому было проведено начисление.
  • Дата заезда - дата заезда гостя.
  • День заезда - день заезда гостя.
  • Час заезда - час заезда гостя.
  • Месяц заезда - месяц заезда гостя.
  • Квартал заезда - квартал заезда гостя.
  • Неделя заезда - неделя заезда гостя.
  • Год заезда - год заезда гостя.
  • Клиент – гость, элемент справочника «Клиенты».
  • Тип клиента - значение типа клиента. Ссылка на справочник «Типы клиентов».
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает контрагента, на которого были начислены услуги (контрагент из «Фолио»). Если гость оплачивал на стойке как «физическое лицо», то значение контрагента в данном отчет будет пустым.
  • Скидка - значение скидки в процентах.
  • Дисконтная карта - ссылка на справочник дисконтных карт. Заполняется при поселении, если гость предъявил карту.
  • Тип скидки - разделитель учета, ссылка на справочник «Тип скидок». Установка соответствия между набором услуг и размером скидки.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой-либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Путевка - ссылка на путевку или курсовку по которой поселили гостя.
  • Направление маркетинга - ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Документ основание – ссылка на документ «Размещение», «Бронь», «Бронь ресурса» или «Состав квоты» по которому были начислены услуги. Если в отчете ссылка на «Филио», то это означает, что услуги были начислены по счету «стороннему посетителю» не проживающему в гостинице.
  • Способ оплаты счета.
  • Отчетная валюта – валюта, указанная в качестве отчетной в справочнике «Гостиницы».
  • Номер - номер комнаты, в который был поселен гость.
  • Цена проживания - среднее значение стоимости номера за интересуемый период.
  • Квота номеров - значение квоты номеров, по которой был поселен гость.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип тарифа - разделитель учета, в разрезе типа тарифа можно строить аналитические отчеты.
  • Тип номера - ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).
  • Услуга – указывается услуга, для которой было начисление.
  • Источник информации - ссылка на аналитический справочник «Источник информации о гостинице». Используется для анализа рекламных источников. Данное значение может быть обязательным полем для заполнения при поселении.
  • Цель поездки - ссылка на справочник «Цели поездки». Используется для анализа. Может использоваться в группировках и отборах. Данное значение в «Размещении» может быть обязательным для заполнения полем.

Колонки

  • Продано дополнительных мест – количество дополнительных место-дней, проданных за период отчета.
  • Продано дополнительных мест по проживанию - количество дополнительных место-дней, проданных по размещению гостя «от стойки».
  • Продано дополнительных мест по брони - количество дополнительных место-дней, проданных «по брони».
  • Продано мест – количество место-дней, проданных за период отчета.
  • Продано мест по проживанию - количество дополнительных мест, проданных по размещению гостя «от стойки».
  • Продано мест по брони - количество дополнительных мест, проданных «по брони».
  • Комиссия - сумма комиссии за период отчета (комиссия по проживанию + Комиссия по брони).
  • Комиссия без НДС - та же сумма без НДС.
  • Комиссия по проживанию - сумма комиссии, рассчитанная по размещениям гостей «от стойки».
  • Комиссия без НДС по проживанию - та же сумма без НДС.
  • Комиссия по брони - сумма комиссии, рассчитанная по размещениям гостей «по брони».
  • Комиссия без НДС по брони - та же сумма без НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – та же сумма без НДС.
  • Сумма скидки по проживанию - сумма скидки, рассчитанная по размещениям гостей «от стойки».
  • Сумма скидки по проживанию без НДС - та же сумма без НДС.
  • Сумма скидки по брони – сумма скидки, рассчитанная по размещениям гостей «по брони».
  • Сумма скидки по брони без НДС - та же сумма без НДС.
  • Человеко-дни – количество человеко-дней за период отчета. Рассчитывается по каждому гостю независимо от стоимости проживания. Если гостя поселили на 3 дня, то в данную колонку добавится 3.
  • Человеко-дни по проживанию - количество человеко-дней, проданных по размещению гостя «от стойки».
  • Человеко-дни по брони - количество человеко-дней, проданных по размещению гостя «по брони».
  • Заезд гостей – количество заехавших за период отчета гостей.
  • Заезд гостей по проживанию - количество гостей, заехавших по размещению «от стойки».
  • Заезд гостей по брони - количество гостей, заехавших «по брони».
  • Доходы от продажи номеров – сумма только от проживания гостя, без учета дополнительных услуг. Скидки учтены.
  • Доходы от продажи номеров без НДС – та же сумма без НДС.
  • Доходы от продажи номеров по проживанию - сумма от проживания гостей, если они заехали «от стойки».
  • Доходы от продажи номеров по проживанию без НДС - та же сумма без НДС.
  • Доходы от продажи номеров по брони - сумма от проживания гостей, если они заехали «по брони».
  • Доходы от продажи номеров по брони без НДС - та же сумма без НДС.
  • Продано номеров – количество номеро-дней за период отчета.
  • Продано номеров по проживанию - количество номеро-дней, проданных по размещениям гостей в гостинице «от стойки».
  • Продано номеров по брони - количество номеро-дней, проданных «по брони».
  • Продажи – сумма, начисленная по всем услугам, включая дополнительные.
  • Продажи без НДС – та же сумма без НДС.
  • Продажи по проживанию - сумма, начисленная по всем услугам, включая дополнительные по размещениям «от стойки».
  • Продажи по проживанию без НДС - та же сумма без НДС.
  • Продажи по брони - сумма, начисленная по всем услугам «по брони».
  • Продажи по брони без НДС - та же сумма без НДС.
