Отчет "Продажи квот номеров"

Назначение

Имя отчета в конфигураторе «RoomQuotaSales».

Отчет используется для анализа продаж квот номеров. Отчет показывает по датам сколько всего было номреов в квоте, сколько на эту дату из квоты номеров забронировано, сколько уже размещено и сколько номеров осталось в квоте.

Отборы на форме отчета

  • По типу номера - задает отбор по набору услуг.
  • По номеру - задает отбор по номеру, если поле оставить пустым, то будут выведенные данные по всем номерам гостиницы.
  • По контрагенту - если поле оставить пустым, то программа сформирует отчет по всем контрагентам.
  • По агенту.
  • По договору - отбор по номеру договора.

Группировки

  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает на организацию, юр. лицо, от имени которого было произведено бронирование (размещение).
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Период - дата продажи квоты.
  • Номер – указание номера комнаты.
  • Квота номеров – ссылка на справочник «Квоты номеров».
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).

Колонки

  • Всего мест в квоте – количество мест в текущей квоте.
  • Мест осталось – невыбранный остаток мест в квоте.
  • Мест забронировано – число забронированных мест в квоте.
  • Гарантированная бронь мест – число гарантированных данной квотой мест.
  • Гарантированная бронь номеров – число гарантированных данной квотой номеров.
  • Мест занято – число мест, на которые в данной квоте прошло размещение.
  • Номеров занято – число номеров, на которые в данной квоте прошло размещение.
  • Всего номеров в квоте – кол-во номеров в текущей квоте.
  • Номеров осталось – невыбранный остаток номеров в квоте.
  • Номеров забронировано – число забронированных номеров в квоте.
