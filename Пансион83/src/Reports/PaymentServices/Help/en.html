Отчет "Распределение платежей по услугам"

Назначение

Имя отчета в конфигураторе «PaymentServices».

Отчет позволяет распечатать распределение платежей по оказанным услугам. Используется при включенном в параметрах гостиницы режиме автоматического распределения платежей по оказываемым услугам.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета оплаты - по способу оплаты, по кассовой секции, по ККМ.
  • По фирме - задает отбор по фирме, если поле оставить пустым, то будут выведенные данные по всем фирмам.

Группировки

  • Учетная дата – дата платежа (без указания времени).
  • Учетный месяц – номер месяца платежа.
  • Учетный квартал – номер квартала.
  • Учетная неделя – номер недели платежа.
  • Учетный год – год платежа.
  • Лицевой счет – ссылка на документ «Фолио».
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Клиент – гость, ссылка на справочник «Клиенты».
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – основание для расчетов между юр.лицом гостиницы и контрагентом.
  • Валюта фолио – валюта документа «Лицевой счет».
  • Контрагент – юр.лицо, заключившее с гостиницей договор, на основании которого произведено размещение гостя.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Документ основание - документ «Размещение».
  • Номер – указание номера, занимаемого клиентом.
  • Платеж – ссылка на документ «Платеж».
  • Автор.
  • ККМ – указывается ККМ, через которую прошел платеж (в случае наличной оплаты).
  • Плательщик – гость или контрагент.
  • Способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществляется оплата текущего гостя.
  • Кассовая секция – номер секции ККМ.
  • Услуга – услуга, на которую распределился данный платеж.

Колонки

  • Дата заезда – указание даты зезда клиента.
  • Дата выезда - указание даты выезда клиента.
  • Долг на конец периода – долг по оплате данной услуги после на момент конца периода, указанного в отчете.
  • Сумма платежей за период – общий платеж по данной услуге за указанный период.
  • Долг на начало периода – долг по оплате данной услуги после на момент начала периода, указанного в отчете.
  • Сумма начислений по услугам за период – сумма начисления по данной услуге за указанный период.
