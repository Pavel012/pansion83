Отчет "График рабочего времени сотрудников"

Назначение

Имя отчета в конфигураторе «EmployeeWorkingTimeSchedule».

Отчет позволяет распечатать графики рабочего времени сотрудников за указанный период.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета выполнения работ – по сотруднику, отделу, секции номеров.
  • По номеру - задает отбор по номеру. Если поле оставить пустым, то будут выведенные данные по всем номерам.

Группировки

  • Учетная дата – дата работы сотрудника.
  • Учетный месяц – номер месяца.
  • Учетный квартал – номер квартала.
  • Учетная неделя – номер недели в году.
  • Учетный год – год выполнения работы.
  • Отдел – ссылка на справочник Отделы, указывает принадлежность сотрудника какому – либо разделу.
  • Сотрудник – ссылка на справочник «Сотрудники».
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Количество рабочих часов – отработанное сотрудником время.
  • Секция номеров – ссылка на справочник «Секции номерного фонда».
  • Номер – указание номера, в котором производится работа.
  • Тип дня календаря – ссылка на справочник «Типы дней календаря», может использоваться в аналитических отчетах.
  • Расписание дня - ссылка на справочник «Расписания», может использоваться в аналитических отчетах.
