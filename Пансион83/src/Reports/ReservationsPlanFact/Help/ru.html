Отчет "План-фактный анализ брони"

Назначение

Имя отчета в конфигураторе «ReservationsPlanFact».

Отчет позволяет получить таблицу сравнения итогов заезда гостей по оформленной в программе брони и по реальным размещениям.

Отборы на форме отчета

  • По типу номера - задает отбор по типу номера, если поле оставить пустым, то будут выведены данные по всем типам номеров гостиницы.

Группировки

  • Размещение - ссылка на документ «Размещение».
  • Статус размещения – статус данного размещения.
  • Вид размещения – вид размещения, соответствующий текущему размещению.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • День заезда - день заезда по брони или размещению.
  • Месяц заезда - номер месяца.
  • Квартал заезда - номер квартала заезда по брони.
  • Неделя заезда - номер недели.
  • Год заезда - год заезда по брони.
  • Договор – основание для расчетов между юр.лицом гостиницы и контрагентом.
  • Контрагент – юр.лицо, заключившее с гостиницей договор, на основании которого произведено размещение гостя.
  • Гость – гость, справочник «Клиенты».
  • Группа гостей – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Документ основание - ссылка на документ «Бронь».
  • Бронь - ссылка на документ «Бронь».
  • Статус брони – статус текущей брони.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором планируется размещение гостя.

Колонки

  • Заезд доп. мест - количество заехавших гостей с размещением «Доп. место».
  • Бронь доп. мест - количество забронированных дополнительных мест.
  • Заезд мест - количество заехавших гостей с размещением «Место».
  • Бронь мест - количество забронированных мест.
  • Заезд гостей - количество заехавших гостей.
  • Бронь гостей - количество гостей на вид размещения по брони.
