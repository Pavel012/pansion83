Отчет "Реестр использованных бланков путевок"

Назначение

Имя отчета в конфигураторе «HotelProductsRegister».

Отчет используется для формирования списка номеров использованных (в том числе и испорченных) бланков путевок. Отчет также позволяет показать реестр отгруженных сторонним контрагентам бланков.

Отборы на форме отчета

  • По периоду - задает отбор по дате создания путевки в справочнике путевок или дате отгрузки бланка путевки контрагенту.
  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По типу путевки - задает отбор по типу путевок/курсовок.
  • Показывать использованные путевки - распечатываются не помеченные на удаления номера путевок из справочника путевок, т.е. путевки указанные в брони и размещениях.
  • Показывать испорченные путевки - распечатываются помеченные на удаление номера путевок из справочника путевок.
  • Показывать отгруженные но не использованные путевки - распечатываются номера бланков путевок отгруженных сторонним контрагентам, по которым еще не оформлялись размещения (не создавалась путевка в справочнике путевок).

Группировки

  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Тип путевки – тип путевки/курсовки.
  • Валюта путевки – валюта стоимости путевки.
  • Квота номеров – квота номеров, соответствующая данной путевке/курсовке.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера путевки. 

Колонки

  • Стоимость путевки – общая сумма продаж услуг по путевке/курсовке.
  • Цена путевки – стоимость путевки указанная в карточке путевке (плановая стоимость).
  • Оплата наличными – сумма всех наличных платежей по лицевым счетам где встречаются услуги начисленные по путевке.
  • Оплата кред. картой – сумма всех платежей по кредитным картам по лицевым счетам где встречаются услуги начисленные по путевке.
  • Оплата б/н – сумма всех банковских переводов на расчетный счет гостиницы по лицевым счетам где встречаются услуги начисленные по путевке.
