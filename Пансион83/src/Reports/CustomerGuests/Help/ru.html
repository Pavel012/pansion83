Отчет "Гости от контрагентов"

Назначение

Имя отчета в конфигураторе «CustomerGuests».

Отчет позволяет получить список гостей от контрагентов с выручкой по каждому гостю. В отчете показываются гости по контрагентам, который заполняется из документа «Размещение» независимо от способа оплаты. Т.е. гость в этом отчете будет "привязан" к контрагенту как в случае, когда в размещении указан контрагент, но гость оплачивает свое проживание сам наличными и в бухгалтерии его услуги отражаются как услуги оказанные физ. лицам, так и в случае когда контрагент является плательщиком по услугам гостя и ему формируются акты об оказании услуг.

Отчет предназначен для проведения сверок с контрагентами или агентами по фамилиям и срокам проживания гостей.

Отчет показывает только суммы всех оказанных услуг и отдельно суммы за проживание. Отчет не позволяет "разворачивать" суммы по услугам. Для того, чтобы распечатать список гостей оплаченных контрагентом с детализацией по услугам необходимо использовать отчет "Продажи по контрагентам по услугам".

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По фирме - если указано, то в отчет будут выведены только данные указанной фирмы. Если реквизит пустой, то будут выведены данные по всем фирмам.
  • По агентам.
  • По наборам услуг.
  • По основным разрезам учета взаиморасчетов – по контрагенту, договору и группе гостей.
  • По типу отбора периода – отчет позволяет отбирать услуги либо по датам начисления услуг, либо по датам заезда или выезда гостей. В последних двух случаях отчет показывает суммы по услугам за весь период проживания гостя. 

Группировки

Гостей можно группировать по контрагентам, агентам, договорам, тарифам, номеру комнаты, группе, типу номера и ресурсу.

Чаще всего используется группировка по контрагенту и номеру группы или номеру комнаты.

  • Вид размещения – ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Клиент – ссылка на справочник «Клиентов», значение гостя, которого поселили.
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – договор контрагента, значение из документа «Размещение».
  • Контрагент – ссылка на справочник контрагентов, заполняется из документа «Размещение» независимо от способа оплаты.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой-либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Документ основание – ссылка на документ «Размещение» или «Бронь ресурсов», тот документ на основании которого было сформировано начисление. Если в качестве документа основания указано «Фолио», то это означает что начисление было оформлено как «стороннему посетителю, т.е. без поселения.
  • Отчетная валюта – валюта, указанная в качестве отчетной в справочнике «Гостиницы», все суммы пересчитываются в отчетную валюту в момент начисления для целей отчетности.
  • Ресурс – ссылка на справочник «Ресурсы», ресурс, задействованный в оказании услуги.
  • Номер – номер комнаты, в которой проживал гость.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).
  • Статус – значение статуса Брони, Размещения или Брони ресурсов.

Колонки

  • Продано дополнительных мест – количество доп. место-дней, проданных в данной группе.
  • Продано мест – количество место-дней по гостю.
  • Дата заезда – дата и время заезда данной группы гостей.
  • Дата выезда – дата и время выезда данной группы гостей.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма комиссии, которая начислена агенту за вычетом суммы НДС.
  • Сумма скидки – значение скидки по отношению к тарифу размещения.
  • Сумма скидки без НДС – значение скидки по отношению к тарифу размещения за вычетом суммы НДС.
  • Человеко-дни – количество человеко-дней. Рассчитывается по каждому гостю независимо от стоимости проживания. Если гостя поселили на 3 дня, то в данную колонку добавиться 3, если период проживания попадает в период отчета.
  • Заезд гостей – количество заехавших гостей за период, указанный в отчете.
  • Доходы от продажи номеров – сумма только от проживания гостя, без учета дополнительных услуг. Скидки учтены.
  • Доходы от продажи номеров без НДС – предыдущая сумма за вычетом НДС.
  • Продано номеров – количество проданных номеро-дней (или «ночей») по гостю за период отчета.
  • Продажи – сумма, начисленная за все услуги по гостю или по счету, включая дополнительные услуги.
  • Продажи без НДС – предыдущая сумма продаж за вычетом НДС.
