Отчет "Обороты продаж по гостям от контрагентов"

Назначение

Имя отчета в конфигураторе «CustomerGuestsTurnovers».

Отчет позволяет получить итоги по продажам услуг в разрезе измерений регистра «Продажи». Отчет формируется по виртуальной таблице регистра и предназначен для быстрого получения итогов продаж в любом разрезе учета. С помощью данного отчета можно получать итоги с точностью до гостя, например, формировать список гостей отсортированных по убыванию выручки, которую они "принесли" в гостиницу.

В отличие от отчета "Обороты продаж по контрагентам" этот отчет показывает обороты продаж не только по услугам, которые оплачивались самими контрагентами, но и обороты продаж по услугам гостей, у которых в размещении был указан контрагент, а оплачивали услуги они сами. Другими словами, для того чтобы услуга попала "под" контрагента в отчете "Обороты продаж по контрагентам" требуется, чтобы услуга была начислена на лицевой счет контрагента. Для того чтобы услуга попала "под" контрагента в отчете "Обороты продаж по гостям от контрагентов" достаточно, чтобы контрагент был указан в размещении, по которому проживал гость. Лицевой счет услуги при этом может быть любым, в том числе оплачиваемым самим гостем. 

Если указан период отчета в будущем, то отчет учитывает планируемые продажи по действующей брони.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета взаиморасчетов – по контрагенту, договору и группе гостей.

Группировки

  • Учетная дата – дата, за которую подводятся итоги продаж (без указания времени).
  • Учетный месяц – месяц, за который подводятся итоги продаж. Поле содержит числовое значение месяца, например март = 3. Поле предназначено для группировки и получения данных по месяцам.
  • Учетный квартал – номер квартала учетной даты, поле предназначено для группировки данных отчета.
  • Учетная неделя – номер недели в году, на которую приходится учетная дата. Поле предназначено для группировки данных по неделям.
  • Учетный год – числовое значение года учетной даты.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Клиент – гость, элемент справочника «Клиенты».
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – ссылка на договор контрагента указанный в размещении или брони ресурсов, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает контрагента из размещения или брони ресурсов.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой-либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Документ основание – ссылка на документ «Размещение» или «Бронь ресурса» по которому были начислены услуги. Если в отчете ссылка на «Фолио», то это означает, что услуги были начислены по лицевому счету «стороннего посетителя» не проживающего в гостинице.
  • Способ оплаты из лицевого счета.
  • Отчетная валюта – валюта, указанная в качестве отчетной в справочнике «Гостиницы».
  • Цена проживания - среднее значение стоимости номера за интересуемый период.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Услуга – указывается услуга, для которой было начисление.

Колонки

  • Продано дополнительных мест – количество дополнительных место-дней, проданных за период отчета.
  • Продано мест – количество место-дней, проданных за период отчета.
  • Дата первого заезда клиента - имеет смысл, когда в группировке есть клиент.
  • Дата последнего заезда клиента - имеет смысл, когда в группировке есть клиент.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма комиссии без НДС.
  • Дата первого заезда контрагента - имеет смысл, когда в группировке есть контрагент.
  • Дата последнего заезда контрагента - имеет смысл, когда в группировке есть контрагент.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – та же сумма без НДС.
  • Человеко-дни – количество человеко-дней за период отчета. Рассчитывается по каждому гостю независимо от стоимости проживания. Если гостя поселили на 3 дня, то в данную колонку добавится 3.
  • Заезд гостей – количество заехавших за период отчета гостей.
  • Доходы от продажи номеров – сумма только от проживания гостя, без учета дополнительных услуг. Скидки учтены.
  • Доходы от продажи номеров без НДС – та же сумма за вычетом НДС.
  • Продано номеров – количество номеродней за период отчета.
  • Продажи – сумма, начисленная по всем услугам, включая дополнительные.
  • Продажи без НДС – та же сумма без НДС.
