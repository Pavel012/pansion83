Отчет "Доходность и загрузка"

Назначение

Имя отчета в конфигураторе «RevenueTurnovers».

Отчет позволяет получить итоги по продажам номеров/мест в разрезе периодов (дней, недель, месяцев, кварталов, годов). По каждому периоду рассчитываются проценты продаж.

Если указать период отчета в будущем, то отчет будет учитывать планируемые продажи по брони.

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По услуге - задает отбор по услуге, если поле оставить пустым, то будут выведенные данные по всем номерам гостиницы.
  • По номеру.
  • По типу номера.

Отчет всегда группирует данные по периодам: дням, неделям, месяцам, кварталам, годам. Тип периода выбирается на форме отчета.

Колонки

  • Средняя цена места в номере (ADBR) – усредненная по периоду отчета цена места в номере.
  • Средняя цена места в номере (ADBR)без НДС – усредненная по периоду отчета цена места в номере за вычетом НДС.
  • Продано дополнительных мест – число проданных за период отчета дополнительных мест.
  • Средняя цена номера (ADR) – усредненная по периоду отчета цена номера.
  • Средняя цена номера (ADR)без НДС – усредненная по периоду отчета цена номера за вычетом НДС.
  • Средняя продолжительность проживания гостя (ALS) – средняя в данном периоде продолжительность проживания гостя.
  • Продано мест – число проданных в данном периоде мест.
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма вычисленной комиссии для агента за вычетом НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – сумма предоставленной скидки за вычетом НДС.
  • Человеко-дни - количество человек, проживающих в гостинице на отчетную дату.
  • Заезд гостей – число гостей, заехавших в данный период.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Период – указывает текущий период; в настройке отчета можно выбрать вид периода – день, неделя, месяц, год.
  • Представление периода - период отчта разбивается по дням.
  • Отчетная валюта - валюта, указанная в качестве отчетной для данной гостиницы.
  • Средний доход с доступных к продаже мест (Rev. PAB) – отношение общего дохода с доступных к продаже мест в номерах данного типа к кол-ву доступных к продаже мест номерах данного типа.
  • Средний доход с доступных к продаже мест (Rev. PAB) без НДС – та же сумма без НДС.
  • Средний доход с клиента за ночь (Rev. PAC) – отношение общей суммы продаж за период к общему числу человеко-дней.
  • Средний доход с клиента за ночь (Rev. PAC) без НДС – та же сумма без НДС.
  • Средний доход с доступных к продаже номеров (Rev. PAR) отношение общего дохода с доступных к продаже номеров данного типа к количеству доступных данного типа.
  • Средний доход с доступных к продаже номеров (Rev. PAR) без НДС – та же сумма без НДС.
  • Доходы от продажи номеров – общая сумма продаж услуг проживания в данном периоде.
  • Доходы от продажи номеров без НДС – общая сумма продаж услуг проживания за вычетом НДС в данном периоде.
  • Продано номеров – общее количество проданных в периоде номеров.
  • Продажи – общая сумма продаж (проживание и другие услуги) в данном периоде.
  • Продажи без НДС – общая сумма продаж (проживание и другие услуги) в данном периоде за вычетом НДС.
  • Всего мест – общее количество мест в данном периоде.
  • Всего доступно мест – общее количество доступных мест в данном периоде.
  • Всего заблокировано мест – общее количество заблокированных мест в данном периоде.
  • Общий % загрузки мест – отношение числа проданных мест к числу мест в номерном фонде.
  • Общий % продаж мест – отношение числа проданных мест к числу доступных для продажи мест в номерном фонде.
  • Всего номеров – общее количество номеров в данном периоде.
  • Всего доступно номеров – общее количество доступных номеров в данном периоде, вычисляется как «Всего номеров» минус «Всего заблокировано номеров».
  • Всего заблокировано номеров – общее количество заблокированных номеров в данном периоде.
  • Общий % загрузки номеров – отношение числа проданных номеров к числу номеров в номерном фонде.
  • Общий % продаж номеров – отношение числа проданных номеров к числу доступных для продажи номеров в номерном фонде ("Всего номеров" в номерном фонде минус число "Заблокированных номеров").
