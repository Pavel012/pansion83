Отчет "Гости от контрагентов с услугами"

Назначение

Имя отчета в конфигураторе «CustomerGuestsWithServices».

Отчет во многом совпадает с отчетом «Сводка по гостям от контрагентов», только добавлена детализация по услугам. Т.е. для каждого гостя в отчете показываются итоги продаж по услугам.

Если на форме отчета включить галочку "По дням", то итоги по услугам будут также разбиты по дням проживания, что удобно для выполнения сверок взаиморасчетов по сложным группам.

Отчет считает, что гость приехал от контрагента, если контрагент указан в карточке размещения или брони, не зависимо от того, кто являлся плательщиком по оказанным услугам.

Отчет используется для формирования реестров гостей приехавших от контрагентов и может использоваться для формирования базы расчета агентских отчислений.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета взаиморасчетов – по контрагенту, договору и группе гостей.
  • По агентам.

Группировки

  • Вид размещения – ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Учетная дата - дата начисления услуги. Заполняется,если в правом верхнем углу отчета поставлена галочка «По дням».
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Клиент – гость, ссылка на справочник «Клиенты».
  • Фирма – юр.лицо гостиницы, от имени которого оформлено размещение гостя.
  • Договор – договор из фолио, по которому проведено начисление.
  • Контрагент – ссылка на справочник контрагентов, указывает контрагента, на которого были начислены услуги (контрагент из «Фолио»). Если гость оплачивал на стойке как «физическое лицо», то значение контрагента в данном отчет будет пустым.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой-либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Документ основание – ссылка на документ, по которому производился расчет и начисление услуг, обычно принимает значения «Размещение» или «Бронь ресурса». В случае если в колонке указано «Фолио», то начисления проводили для «стороннего посетителя», т.е. клиента не проживающего в гостинице.
  • Отчетная валюта – валюта, указанная в качестве отчетной для данной гостиницы.
  • Ресурс – ресурс, задействованный в оказании услуги.
  • Номер – номер комнаты, в которой разместили клиента.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость.
  • Услуга – услуга, начисленная клиенту.
  • Статус – отражает статус размещения на момент формирования отчета, может быть «Заезд», «Выезд» или другое значение из справочника «Статусы размещений».

    Колонки

    • Продано дополнительных мест – количество место-дней за период отчета.
    • Продано мест – количество место-дней, начисленных за период отчета.
    • Дата заезда – указание даты и времени заезда гостя.
    • Дата выезда – указание даты и времени выезда заезда.
    • Комиссия – сумма вычисленной комиссии для агента.
    • Комиссия без НДС – та же сумма без НДС.
    • Сумма скидки – сумма предоставленной скидки.
    • Сумма скидки без НДС – та же сумма без НДС.
    • Человеко-дни – количество человеко-дней за период отчета. Рассчитывается по каждому гостю независимо от стоимости проживания.
    • Заезд гостей – количество заехавших за период отчета гостей.
    • Доходы от продажи номеров – сумма, начисленная за услуги проживания клиента за период отчета.
    • Доходы от продажи номеров без НДС – та же сумма без НДС.
    • Продано номеров – кол-во номеродней начисленных за период отчета.
    • Продажи – сумма, начисленная клиенту за период отчета.
    • Продажи без НДС – та же сумма без НДС.
