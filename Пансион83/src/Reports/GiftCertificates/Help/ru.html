Отчет "Действующие подарочные сертификаты"

Назначение

Имя отчета в конфигураторе «ActiveGiftCertificates».

Отчет формируется по данным регистра «Подарочные сертификаты» (GiftCertificates). Отчет предназначен для получения списка действующих, не заблокированных на дату отчета подарочных сертификатов, по которым не полностью использована сумма сертификата. Отчет выводит количество таких сертификатов и баланс остатка средств на каждом сертификате.

Отборы на форме отчета

  • Дата отчета – задает дату и время, на которые получаются остатки не использованных сумм сертификатов. Если этот параметр пустой, то остатки берутся на конец времен.
  • Услуга – задает отбор по услуге подарочного сертификата. Можно указывать папку услуг или оставить пустой.
  • Гостиница – задет отбор по гостинице. Можно указывать папку гостиниц или оставить пустой.

Группировки

  • Клиент – покупатель сертификата.
  • Контрагент – Организация, которая оплатила покупку сертификата.
  • Договор - Договор с организацией, которая оплатила покупку сертификата.
  • Группа – номер группы, по которой проживал клиент - покупатель сертификата.
  • Фирма - Юридическое лицо отеля, по которому прошла продажа сертификата.
  • Номер - номер комнаты, где проживал клиент - покупатель сертификата.
  • Тип номера - категория комнаты, где проживал клиент - покупатель сертификата.
  • Услуга - услуга продажи сертификата
  • Документ основание - размещение/бронь/бронь ресурса клиента - покупателя сертификата.

Колонки

  • Подарочный сертификат - код (идентификатор) подарочного сертификата.
  • Начало периода, Конец периода - Период проживания клиента - покупателя сертификата.
  • Баланс суммы сертификата на дату и время отчета – не использованная для оплаты услуг сумма сертификата.
  • Количество - количество сертификатов в отчете.
