Отчет "Обороты продаж по регионам"

Назначение

Имя отчета в конфигураторе «GeoSalesTurnovers».

Отчет позволяет получить итоги по продажам номеров в разрезе стран, регионов, областей, городов, откуда приехали гости или в разрезе возраста гостей по регистру «Продажи». 

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По стране - задает отбор по стране. Если поле оставить пустым, то будут выведенные данные по всем странам.
  • По цели визита - задает отбор по цели визита гостя.
  • По источнику информации.
  • По набору услуг - задает отбор по набору услуг.

Группировки

  • Учетная дата – дата продажи услуги (без указания времени).
  • Учетный месяц - номер месяца продажи услуги.
  • Учетный квартал - номер квартала продажи услуги.
  • Учетная неделя - номер недели продажи услуги.
  • Учетный год - год продажи услуги.
  • Возрастная группа.
  • Возраст гостя.
  • Гражданство гостя.
  • Город гостя – город, из которого приехал гость.
  • Регион гостя – регион, из которого приехал гость.
  • Фирма – юр. лицо гостиницы, от имени которой оформлено размещение гостя.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Документ основание – ссылка на документ «Размещение» или «Бронь».
  • Отчетная валюта – валюта, указанная в качестве отчетной для данной гостиницы.
  • Услуга – проданная услуга.
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.
  • Цель поездки – ссылка на справочник «Цели визита», может использоваться для построения аналитических отчетов.

Колонки

  • Продано дополнительных мест – число дополнительных мест, проданных в данной группе гостей.
  • Средняя цена места - отношение доходов от продажи номеров к количеству проданных мест за учетный период.
  • Средняя цена места без НДС - та же сумма без НДС.
  • Средняя цена номера - отношение доходов от продажи номеров к количеству проданных номеров за учетный период.
  • Средняя цена номера без НДС - та же сумма без НДС.
  • Ремонт мест - количество мест в гостинице, находящихся «на ремонте».
  • Всего мест - общее количество мест в гостинице.
  • Продано мест – число мест, проданных в данной группе гостей.
  • % Загрузки мест - отношение числа проданных мест к общему количеству мест (Всего мест) * 100%.
  • % Продаж мест - отношение числа проданных мест к доступному для продажи количеству мест (за ислючением мест, находящихся на ремонте, заблокированных) * 100%.
  • Комиссия – сумма начисленной комиссии.
  • Комиссия без НДС – та же сумма без НДС.
  • Сумма скидки – скидка по отношению к тарифу размещения.
  • Сумма скидки без НДС – скидка за вычетом НДС.
  • Человеко-дни – количество человеко-дней. Рассчитывается по каждому гостю независимо от стоимости проживания.
  • Заезд гостей – количество заехавших за период отчета гостей.
  • Средний доход с места (RevPAB) - отношение общего дохода с проданых мест в номерах гостиницы к общему количеству мест (Всего мест).
  • Средний доход с места (RevPAB) без НДС - та же сумма, но без учета НДС.
  • Средний доход с номера (RevPAR) - отношение общего дохода с проданых номеров к общему числу номеров в гостинице (Всего номеров).
  • Средний доход с номера (RevPAR) без НДС - та же сумма, но без учета НДС.
  • Доходы от продажи номеров – сумма только от проживания гостя, без учета дополнительных услуг. Скидки учтены.
  • Ремонт номеров - количество номеров, находящихся на ремонте.
  • Всего номеров - общее количество номеров в гостинице.
  • Продано номеров – количество проданных в данной группе номеров.
  • % Загрузки номеров - отношение числа проданных номеров к общему количеству номеров (Всего номеров) * 100%.
  • % Продаж номеров - отношение числа проданных номеров к количеству номеров, доступных к продаже (исключая заблокированные номера) * 100%.
  • Продажи – сумма продаж всех услуг по данной группе, включая дополнительные.
  • Продажи без НДС – та же сумма без НДС.
  • Общие продажи - сумма продаж по всем регионам.
  • Общий % продаж - отношение Продажи/Общие продажи
