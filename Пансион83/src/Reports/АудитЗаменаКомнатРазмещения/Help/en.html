Отчет "Аудит замен номеров комнат в размещениях"

Назначение

Имя отчета в конфигураторе «AccommodationChangeRoomAudit».

Отчет используется для аудита операций по замене номера в размещении. Основная опасность операции в том, что заменив номер портье может выписать электронный ключ и затем провести замену номера обратно, на тот номер где гость на самом деле проживает. В результате портье получает возможность выписывать гостевой ключ в номер, где никто не проживает.

Отборы на форме отчета

  • Период отчета – отбирает за какой период сформировать отчет.
  • Гостиница – задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • Номер – задает отбор по номеру комнаты или по группе номеров.
  • Тип номера – задает отбор по типу номера или по группе типов номеров.
  • Сотрудник – задает отбор по сотруднику, проведшему аннулирование.

Группировки

  • Размещение - ссылка на справочник «Размещения», указывает на текущее размещение.
  • Вид размещения — ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Автор – сотрудник, оформивший гостя (пользователь, создавший документ «Размещение»).
  • Номер куда заехал - номер, куда заехал гость изначально.
  • Гость – ссылка на справочник «Клиентов», значение гостя на данный момент, которого поселили.
  • Автор изменений - сотрудник, проведший изменение размещения.
  • Номер где проживал – номер комнаты, в который был поселен гость первоначально.
  • Номер – номер комнаты, в который поселен гость на данный момент.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость на данный момент.

Колонки

  • Дата заезда – дата и время заезда гостя.
  • Время заселения - дата и время заселения изначально
  • Дата выезда – значение даты и времени выезда.
  • Продолжительность – указывается в днях или часах, в зависимости от выбранного тарифа. Если в тарифе цена указана за сутки, то продолжительность рассчитывается в днях. Если цена указывается за час, то продолжительность рассчитывается в часах.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя на данный момент. Каждый гость обязательно принадлежит какой – либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость на данный момент. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Статус размещения в изменении - статус размещения на момент изменения.
  • Вид размещения в изменении - вид размещения на момент изменения.
  • Агент в изменении – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя на момент изменения. Может также использоваться для аналитики работы агентов или туроператоров.
  • Автомобиль в изменении – строковое значение поля «Автомобиль» из размещения, которое заполняется вручную на момент изменения.
  • Дата заезда в изменении - дата заезда на момент изменения.
  • Дата выезда в изменении - дата выезда на момент изменения.
  • Тип клиента в изменении – значение типа клиента на момент изменения. Ссылка на справочник «Типы клиентов».
  • Фирма в изменении - название фирмы на момент изменения.
  • Контактное лицо в изменении – ссылка на одноименное текстовое поле в документе «Размещение» на момент изменения, указывает на лицо, бронировавшее данный номер от имени клиента. Обычно указывается при бронировании для контрагентов.
  • Договор в изменении – значение договора из «Размещения» на момент изменения.
  • Количество замен номеров - указывает количество раз, когда номер был изменен.
  • Контрагент в изменении – ссылка на справочник «Контрагенты», от имени которого приехал гость на момент изменения. От имени этого контрагента обычно и было оформлено бронирование.
  • Скидка в изменении – значение скидки в процентах на момент изменения.
  • Тип скидки в изменении - значение типа скидки на момент изменения.
  • Продолжительность в изменении – указывается в днях или часах, в зависимости от выбранного тарифа на момент изменения. Если в тарифе цена указана за сутки, то продолжительность рассчитывается в днях. Если цена указывается за час, то продолжительность рассчитывается в часах.
  • Гость в изменении – ссылка на справочник «Клиентов», значение гостя, которого поселили на момент изменения.
  • Проживал в группе - номер группы, до изменения номера.
  • Проживал в гостинице - гостиница, в которой проживал гость до изменения номера.
  • Путевка в изменении – ссылка на путевку или курсовку по которой поселили гостя на момент изменения.
  • Направление маркетинга в изменении – ссылка на элемент справочника «Направления маркетинга» на момент изменения. Используется для анализа, может быть использован в группировках и отборах.
  • Количество человек в изменении - количество человек на момент изменения номера.
  • Планируемый способ оплаты в изменении – значение планируемого способа оплаты в документе «Размещение» на момент изменения. Берется у первого фолио в таблице «Правила начислений». Обычно, если проживание оплачивает контрагент, то значение будет «На расчетный счет».
  • Примечания в изменении – срока, значение из документа «Размещение» на момент изменения.
  • Квота номеров в изменении – значение квоты номеров, по которой был поселен гость.
  • Тариф в изменении – показывает, по какому тарифу поселили гостя, или забронировали номер на момент изменения. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип номера в изменении – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость на момент изменения.
  • Источник информации в изменении – ссылка на аналитический справочник «Источник информации о гостинице». Используется для анализа рекламных источников. Данное значение может быть обязательным полем для заполнения при поселении.
  • Время изменений - премя, когда было проведено изменение.
  • Цель поездки в изменении – ссылка на справочник «Цели поездки». Используется для анализа. Может использоваться в группировках и отборах. Данное значение в «Размещении» может быть обязательным для заполнения полем.
  • Тариф – показывает, по какому тарифу поселили гостя на момент изменения. Тариф определяет цену номера и набор начисляемых услуг.
