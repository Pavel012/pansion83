Отчет "Прайс-лист"

Назначение

Имя отчета в конфигураторе «PriceList».

Отчет позволяет распечатать цены выбранного тарифа в виде прайс-листа гостиницы.

Отборы на форме отчета

  • По гостинице.
  • По основным параметрам ценообразования - по тарифу, по типу дня, по типу клиента.

Группировки

  • Вид размещения – вид размещения, соответствующий текущей цене.
  • Тип дня календаря – какому типу дня соответствует данная цена (высокий/низкий сезон, будни/выходные).
  • Валюта – валюта цены.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).

Колонки

  • Порядок сортировки видов размещений – выводится целое число, показывающее, в каком порядке идут виды размещений.
  • Порядок сортировки типа дня календаря - выводится целое число, показывающее, в каком порядке идут типы дней.
  • Цена по тарифу – текущая цена.
  • Порядок сортировки типов номеров – выводится целое число, показывающее, в каком порядке идут типы номеров.
