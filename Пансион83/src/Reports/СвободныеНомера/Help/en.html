Отчет "Свободные номера"

Назначение

Имя отчета в конфигураторе «VacantRooms».

Отчет выводит список номеров где на указанную дату и время есть хотя бы одно свободное место.

Отчет обычно снимают либо на текущую дату и время, либо на конец текущего дня (т.е. указывают время 23:59:59). В последнем случае отчет показывает сколько и какие номера будут свободны на ближайшую ночь, т.е. сколько гостей с улицы можно будет поселить вечером.

На основании данного отчета часто настраивают отчет "Частично занятые номера". Для этого на форме настроек отчета на закладке "Дополнительные отборы" необходимо добавить условие "Число занятых мест > 0".

Если необходимо показывать только целеком свободные номера, то на форме настроек отчета можно добавить условие "Число свободных номеров > 0".

Отборы на форме отчета

  • Показывать зарезервированные номера - если включен, то в отчет попадут номера, на которые оформлена бронь, но на данный момент номер еще не занят.
  • По типу номера - задает отбор по типу номера в гостинице.
  • По номеру - задает отбор по номеру, если поле оставить пустым, то будут выведенные данные по всем номерам гостиницы.
  • По статусу номера - задает отбор по свободным и/или занятым номерам. В отчете будут отображены номера с выбранным статусом на текущий момент. Если в списке выбора статусов нет нужного, то список следует обновить. Для этого необходимо нажать на крестик справа от кнопки «Статусы номеров». Затем снова нажать на кнопку «Статусы номеров». Программа предложит обновить список.

Группировки

  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Номер – номер, в котором проживает гость.
  • Статус номера – ссылка на справочник «Статусы номеров», указывает статус номера, который обычно меняется при заезде и выселении из номера и обычно принимает значения «Занят», «Свободен», «Грязный», «Чистый».
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).

Колонки

  • Свободно мест – количество свободных мест в данном номере.
  • Дата ближайшего планируемого заезда – если есть резервирование по данному номеру, будет указана дата начала резервирования.
  • Дата освобождения номера – если есть резервирование по данному номеру, будет указана дата конца резервирования.
  • Гость – клиент, ссылка на справочник «Клиенты». Заполняется в том случае, если номер гостем занят не полностью (например, в 2-местном номере поселен с размещением «Место»).
  • Гражданство – гражданство гостя. Заполняется, если поле «Гость» не пустое.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами. Заполняется, если поле «Гость» не пустое.
  • Пол – пол гостя. Заполняется, если поле «Гость» не пустое.
  • Свободно гостей - количество гостей, вычисляемое разницей между тем количеством гостей, которых можно поселить в номере(считая доп. места) и количеством гостей, которые уже поселены в номере.
  • Занято доп. мест – количество занятых дополнительных мест в данном номере.
  • Занято мест – количество занятых мест в данном номере.
  • Гостей – число зарегистрированных гостей в текущем номере.
  • Занято номеров – количество номеров, занятых текущим размещением.
  • Документ основание – документ основание для размещения (Бронь). Заполняется, если размещение было оформлено по брони.
  • Регистратор – ссылка на документ «Размещение». Заполняется, если поле «Гость» не пустое.
  • Свободно номеров – количество свободных номеров.
  • Код типа номера – код из справочника «Типы номеров».
