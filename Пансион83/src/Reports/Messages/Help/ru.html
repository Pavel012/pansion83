Отчет "Задачи"

Назначение

Имя отчета в конфигураторе «Messages».

Отчет распечатывает задачи сохраненные в базе данных. Можно печатать как действующие задачи, так и всю историю задач. 

Отборы на форме отчета

  • По статусу задачи. Если поле оставить пустым, то будут выведенные данные по всем статусам.
  • По сотруднику.
  • По номеру - задает отбор по номеру гостиницы.
  • По отделу.

Группировки

  • Автор – пользователь, создавший сообщение.
  • Кем закрыта - пользователь, закрывший задачу.
  • Дата и время закрытия.
  • Отдел – отдел–адресат сообщения.
  • Сотрудник – сотрудник–адресат сообщения.
  • Задача закрыта - принимает значения «Да» и «Нет».
  • Автор последнего примечания.
  • Статус задачи.
  • Объект – объект базы, являющийся предметом задачи (по умолчанию - номер)

Колонки

  • Количество задач.
  • Последнее примечание - последнее введенное римечание по сообщению.
  • Время последнего примечания.
  • Период – дата и время документа.
  • Регистратор – ссылка на документ «Задача».
  • Текст сообщения.
  • Задача действует с даты.
  • Задача действует по дату – сообщение будет показано в системе в диапазоне между датами «с» - «по».
