Отчет "Продажи путевок"

Назначение

Имя отчета в конфигураторе «HotelProductSales».

Отчет используется для формирования различных реестров по проданным путевкам и курсовкам. Отчет выводит продажи за период так, как это делает любой другой отчет по оказанным услугам. Т.е. отбор услуг идет по их учетной дате и времени оказания.

Отчет отличается от отчета «Обороты продаж по путевкам» тем, что в данном отчете есть колонка "Количество путевок" и отсутствуют колонки "Учетная дата" и "Услуга", т.е отчет не позволяет "развернуть" сумму путевки по оказанным в ней услугам и датам оказания. Также, в отличие от отчета «Обороты продаж по путевкам», по каждой путевке отчет позволяет показать фамилию клиента, период его проживания и категорию комнаты.

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По путевке - задает отбор по типу путевок/курсовок или по конкретной путевке/курсовке.
  • По основным разрезам учета взаиморасчетов - по контрагенту, по договору.
  • По набору услуг - задает отбор по набору услуг.

Группировки

  • Вид размещения – справочник видов размещения.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Клиент – справочник «Клиенты», гость которому была продана путевка.
  • Тип клиента – для указания «бесплатных», «VIP» и т.д. типов гостей.
  • Договор – договор из фолио, по которому проведено начисление.
  • Контрагент – юр.лицо, заключившее с гостиницей договор, на основании которого произведено размещение гостя.
  • Тип скидки – разделитель учета, ссылка на справочник «Тип скидок». Установка соответствия между набором услуг и размером скидки.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Путевка – номер путевки/курсовки.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Гостей – число гостей в данной группе.
  • Планируемый способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществлялась оплата путевки.
  • Отчетная валюта – валюта, обозначенная как отчетная в карточке Гостиницы.
  • Квота номеров – квота номеров, соответствующая данной путевке/курсовке.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг
  • Доходы от продажи номеров – сумма только от проживания гостя, без учета дополнительных услуг. Скидки учтены.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.
  • Цель поездки – ссылка на справочник «Цели визита», может использоваться для построения аналитических отчетов.

Колонки

  • Продано дополнительных мест – количество проданых дополнительных мест.
  • Продано мест – количество мест, проданных в данной группе.
  • Дата заезда – дата заезда данной группы гостей.
  • Продолжительность – продолжительность проживания по путевке.
  • Дата выезда – дата выезда данной группы гостей.
  • Комиссия – сумма начисленной комиссии.
  • Комиссия без НДС – та же сумма за вычетом НДС.
  • Сумма скидки – сумма скидки по отношению к тарифу размещения.
  • Сумма скидки без НДС - та же сумма без НДС.
  • Человеко-дни – количество человеко-дней. Рассчитывается по каждому гостю независимо от стоимости проживания.
  • Заезд гостей – количество заехавших гостей за период отчета.
  • Доходы от продажи номеров - сумма только от проживания гостя, без учета дополнительных услуг. Скидки учтены.
  • Доходы от продажи номеров без НДС – предыдущая сумма без НДС.
  • Продано номеров – количество проданных в данной группе номеров.
  • Продажи – сумма продаж по данной группе.
  • Продажи без НДС – предыдущая сумма без НДС.
  • Кол-во путевок – количество путевок, продажи по которым попали в период отчета
