Отчет "Протокол выполнения работ персоналом"

Назначение

Имя отчета в конфигураторе «EmployeeOperationsHistory».

Отчет позволяет распечатать работы сотрудников за указанный период времени. Если способ учета периода не указан, то отбор выполняется по дате и времени регистрации работы в программе. Если вид учета периода указан, то отбор выполняется по датам и временам начала и завершения работ.

Отчет формируется по данным документов «Регистрация работы сотрудников».

Отборы на форме отчета

  • По гостинице - задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • По основным разрезам учета выполнения работ – по сотруднику, по работе.
  • По номеру - задает отбор по номеру. Если поле оставить пустым, то будут выведенные данные по всем номерам.
  • По типу номера - задает отбор по типу номера. Если поле оставить пустым, то будут выведенные данные по всем номерам.

Группировки

  • Учетная дата – дата регистрации работы в программе (без указания времени).
  • Учетный день – номер дня в месяце.
  • Учетный месяц – номер месяца.
  • Учетный квартал – номер квартала.
  • Учетная неделя – номер недели в году.
  • Учетный год – год выполнения работы.
  • Автор – пользователь, создавший документ «Регистрация работы сотрудников».
  • Сотрудник – сотрудник гостиницы, производивший работу (старший).
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Работа – указание работы, выполненной сотрудником.
  • Код завершения работы в АТС.
  • Код начала работы в АТС.
  • Регистратор – ссылка на документ «Регистрация работы сотрудников».
  • Номер – номер комнаты.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).

Колонки

  • Продолжительность работы в минутах.
  • Количество человек – количество сотрудников, выполняющих работу.
  • Дата и время окончания выполнения работы в номере.
  • Дата и время начала выполнения работы в номере.
  • Период – дата и время начала работы (с точностью до секунд).
  • Примечания – текстовый комментарий.
  • Площадь номера – указание площади в метрах квадратных.
  • Момент времени – ссылка на документ «Регистрация работы сотрудников».
