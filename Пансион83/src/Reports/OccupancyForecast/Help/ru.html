Отчет "Прогноз загрузки"

Назначение

Имя отчета в конфигураторе «OccupancyForecast».

Отчет позволяет по дням формировать прогноз загрузки гостиницы в разрезе типов номеров. Отчет обычно формируется с текущей даты на 14 и 30 дней в будущее. На каждую дату по каждой категории номеров выводится общее число номеров в этой категории, кол-во забронированных номеров по этой категории, кол-во занятых номеров, заезд/выезд номеров, кол-во свободных номеров, заблокированных номеров, номеров в квотах с включенным флажком "Это жесткий блок", планируемые суммы продаж. Отчет используется как службой приема и размещения гостей, так и службой эксплуатации номероного фонда для планирования графиков работы горничных.

Отборы на форме отчета

  • По номеру - обычно указывается группа номеров номерного фонда (корпус или этаж). Если поле оставить пустым, то отчет будет сформирован по всему номерному фонду гостиницы.
  • По типу номера.
  • По набору услуг - задает отбор услуг, по которым рассчитывается сумма оказанных услуг, по набору услуг. Если поле оставить пустым, то сумма планируемых продаж будет посчитана по всем услугам.
  • По услуге - задает отбор услуг, по которым рассчитывается сумма оказанных услуг, по услуге или группе услуг. Если поле оставить пустым, то сумма планируемых продаж будет посчитана по всем услугам.

Группировки колонок

  • По учетной дате - основная группировка, обычно всегда присутствует в отчете
  • По учетной неделе
  • По учетному месяцу
  • По учетному кварталу
  • По учетному году
  • По типу номера - дополнительная группировка, добавляется когда нужна детализация до категорий номеров

Колонки

  • Всего номеров/мест - общее число номеров/мест в гостинице или по категории номеров
  • Номеров/мест на брони - число забронированных номеров/мест на 23:59:59 текущей даты (или последней даты недели, месяца, квартала или года)
  • Номеров/мест на гарантированной брони - число номеров/мест, которые забронированы документами со статусами брони с включенным флажком "Это гарантированная бронь". Рассчитывается аналогично общему числу номеров/мест на брони на конец дня.
  • Номеров/мест на не гарантированной брони - рассчитывается по формуле = "Номеров/мест на брони" - "Номеров/мест на гарантированной брони"
  • Проживает номеров/мест/гостей - рассчитывается по формуле = "Занято номеров/мест/гостей на конец дня" + "Номеров/мест/гостей на брони на конец дня". По смыслу является числом занятых номеров/мест/гостей на конец дня.
  • Занято номеров/мест с учетом брони и квот - рассчитывается по формуле = "Занято номеров/мест на конец дня" + "Номеров/мест на брони на конец дня" + "Кол-во номеров в квотах с включенным флажком "Это жесткий блок" на конец дня". По смыслу дает максимально возможное число занятых номеров в этот день.
  • Прогноз кол-ва гостей с учетом брони - рассчитывается по формуле = "Число проживающих гостей на конец дня" + "Число гостей в брони на конец дня". По смыслу дает максимальное число гостей в номерах на конец дня без учета квот.
  • Номеров/мест в квотах - не проданный остаток номеров/мест в квотах с включенным флажком "Это жесткий блок" на конец дня (23:59:59)
  • Ремонт номеров/мест - кол-во заблокированных номеров/мест на конец дня.
  • Свободно номеров/мест - число свободных номеров/мест на конец дня с учетом брони, проживающих, блокировок и квот с включенным флажком "Это жесткий блок"
  • Заезд номеров/мест/гостей - в прошлом - фактический заезд номеров/мест за день; в будущем - планируемый заезд номеров/мест за день; на текущую дату - планируемый + фактический заезд номеров/мест. Имеется в виду то, что после заезда гостя в номер 1 убирается из планируемого заезда и добавляется к фактическому заезду.
  • Выезд номеров/мест/гостей - в прошлом - фактический выезд номеров/мест за день; в будущем - планируемый выезд номеров/мест за день; на текущую дату - планируемый + фактический выезд номеров/мест. Имеется в виду то, что после выезда гостя из номера 1 убирается из планируемого выезда и добавляется к фактическому выезду.
  • Ожидаемый заезд номеров/мест/гостей - в прошлом - 0 (если нет старой действующей брони); в будущем - планируемый заезд номеров/мест (т.е. совпадает с заездом номеров); на текущую дату - планируемый заезд по тем, кто еще не заехал (т.е. стремится к 0 к вечеру). Ожидаемый заезд = Гарантированный ожидаемый заезд + Не гарантированный ожидаемый заезд
  • Ожидаемый выезд номеров/мест/гостей - в прошлом - 0 (если нет невыселенных размещений); в будущем - планируемый выезд номеров/мест/гостей (т.е. совпадает с выездом номеров); на текущую дату - планируемый выезд по тем, кто еще не выселен (т.е. стремится к 0 к вечеру).
  • Гарантированный ожидаемый заезд номеров/мест/гостей - то же что и просто ожидаемый заезд, но только по гарантированной брони.
  • Не гарантированный ожидаемый заезд номеров/мест/гостей - то же что и просто ожидаемый заезд, но только по не гарантированной брони.
  • Продажи - сумма продаж за день по всем услугам. В будущем включает планируемые продажи по брони. Верна формула Продажи = Продажи по брони + Продажи по проживающим.
  • Продажи без НДС - то же, что и "Продажи" но без суммы НДС
  • Доходы от продажи номеров - сумма продаж за день по услугам проживания.
  • Доходы от продажи номеров без НДС - сумма продаж за день по услугам проживания но без суммы НДС.
  • Продажи по брони - сумма продаж по брони без учета проживающих гостей
  • Продажи по проживающим - сумма продаж по проживающим в гостинице гостям без учета брони
  • Продано номеров/мест - количество проданных номеров/мест по услугам проживания. Совпадает с аналогичными параметрами отчета "Доходность и загрузка"
  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма вычисленной комиссии для агента за вычетом НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – сумма предоставленной скидки за вычетом НДС.
  • Средняя цена места в номере (ADBR) – усредненная за день цена места в номере.
  • Средняя цена места в номере (ADBR) без НДС – усредненнаяза день цена места места в номере за вычетом НДС.
  • Средняя цена номера (ADR) – усредненная за день цена номера.
  • Средняя цена номера (ADR) без НДС – усредненная за день цена номера за вычетом НДС.
  • Средний доход с доступных к продаже мест (Rev. PAB) – отношение общего дохода за проживание в номерах данного типа к кол-ву доступных к продаже мест в номерах данного типа.
  • Средний доход с доступных к продаже мест (Rev. PAB) без НДС – та же сумма без НДС.
  • Средний доход с доступных к продаже номеров (Rev. PAR) - отношение общего дохода запроживание в номерах данного типа к количеству доступных к продаже номеров данного типа.
  • Средний доход с доступных к продаже номеров (Rev. PAR) без НДС – та же сумма без НДС.
  • % занятости номеров/мест – отношение числа занятых + зарезервированных номеров + номеров в квотах с включенным флажком "Это жесткий блок" на конец дня к общему числу номеров (по типу номера) за вычетом заблокированных номеров. В отличие от % продаж считается не по продажам, а по факту занятости номеров.
  • % продаж номеров/мест - отношение числа проданных номеров/мест к общему числу номеров (по типу номера) за вычетом заблокированных номеров. Совпадает с аналогичными параметрами отчета "Доходность и загрузка"
