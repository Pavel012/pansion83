Отчет "Гости, у которых день рождения"

Назначение

Имя отчета в конфигураторе «GuestsWithBirtsDay».

Отчет выводит гостей, у которых в указанном периоде наступит день рождения. Учитываются как проживающие гости, так и гости указанные в брони и которые планируют проживать в указанном на форме отчета периоде.

На форме отчета предусмотрен быстрый отбор по гостинице, номеру (или группе номеров) и типу номера.

Отборы на форме отчета

  • Период отчета – указывает в каком периоде показывать гостей с днями рождения.
  • Гостиница – задает отбор по гостинице, если поле оставить пустым, то будут выведены данные по всем гостиницам.
  • Номер – задает отбор по номеру комнаты или по группе номеров.
  • Тип номера – задает отбор по типу номера или по группе типов номеров.

Группировки

  • Статус – разделитель учета, статус брони или статус размещения.
  • Вид размещения — ссылка на справочник «Виды размещений», указывает на вид текущего размещения гостя.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Автор – сотрудник, оформивший гостя (пользователь, создавший документ «Размещение» или «Бронь»).
  • Учетная дата заезда – дата, когда гость заехал. Не содержит времени заезда. Используется для отбора и группировки.
  • День заезда – день недели заезда гостя. Может использоваться для группировки и отбора.
  • Час заезда – час, когда гость заехал. Может использоваться для группировки и отбора, например, можно получить отчет который покажет распределение количества заехавших гостей по часам.
  • Месяц заезда – месяц заезда гостя, используется для группировки и отборов.
  • Квартал заезда – квартал даты заезда, для группировки и отборов.
  • Неделя заезда – номер недели от даты заезда гостя.
  • Год заезда – значение года заезда гостя.
  • Учетная дата выезда – дата, когда гость выехал, не учитывается время. Используется для отбора и группировки.
  • Тип клиента – значение типа клиента. Ссылка на справочник «Типы клиентов».
  • Контактное лицо – ссылка на одноименное текстовое поле в документе «Размещение» или «Бронь», указывает на лицо, бронировавшее данный номер от имени клиента. Обычно указывается при бронировании для контрагентов.
  • Договор – значение договора из «Размещения» или «Брони».
  • Контрагент – ссылка на справочник «Контрагенты», от имени которого приехал гость. От имени этого контрагента обычно и было оформлено бронирование.
  • Тип контрагента – значение типа контрагента, указанное в «Размещении». Может использоваться для группировки и отбора.
  • Дисконтная карта – ссылка на справочник дисконтных карт. Заполняется при поселении, если гость предъявил карту.
  • Тип скидки – разделитель учета, ссылка на справочник «Тип скидок». Установка соответствия между набором услуг и размером скидки.
  • Гость – ссылка на справочник «Клиентов», значение гостя, которого поселили.
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой-либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостей – количество гостей с днями рождения.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Путевка – ссылка на путевку или курсовку по которой поселили гостя.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Документ основание – документ, на основании которого сделано Размещение. Может быть бронь или другое размещение (в случае переселения). Если поселение «с улицы», то значение поля будет пустым.
  • Планируемый способ оплаты – значение планируемого способа оплаты в документе «Размещение». Берется у первого фолио в таблице «Правила начислений». Обычно, если проживание оплачивает контрагент, то значение будет «На расчетный счет».
  • Регистратор – ссылка на документ «Размещение» или «Бронь» которым был оформлен гость.
  • Примечания – строка с описанием пожеланий гостя 
  • Номер – номер комнаты, в который был поселен гость.
  • Квота номеров – значение квоты номеров, по которой был поселен гость.
  • Тариф – показывает, по какому тарифу поселили гостя, или забронировали номер. Тариф определяет цену номера и набор начисляемых услуг.
  • Тип тарифа – разделитель учета, в разрезе типа тарифа можно строить аналитические отчеты.
  • Номеров – количество номеров занятых данным гостем.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость(планируется размещение).
  • Источник информации – ссылка на аналитический справочник «Источник информации о гостинице». Используется для анализа рекламных источников. Данное значение может быть обязательным полем для заполнения при поселении.
  • Цель поездки – ссылка на справочник «Цели поездки». Используется для анализа. Может использоваться в группировках и отборах. Данное значение в «Размещении» может быть обязательным для заполнения полем.

Колонки

  • Доп. мест – количество дополнительных мест, которые занимает гость.
  • Комиссия агента – значение комиссии агента, начисленной за проживание гостя.
  • Тип комиссии агента – значение типа комиссии, может быть одно из значений перечисления «Виды комиссий агентам». Стандартное значение – «%», процент.
  • Мест – количество мест в номере, занятых гостем.
  • Автомобиль – строковое значение поля «Автомобиль» из размещения, которое заполняется вручную при поселении или при бронировании.
  • Дата заезда – дата и время заезда гостя.
  • Дата выезда – значение даты и времени выезда.
  • Скидка – значение скидки в процентах.
  • Продолжительность – указывается в днях или часах, в зависимости от выбранного тарифа. Если в тарифе цена указана за сутки, то продолжительность рассчитывается в днях. Если цена указывается за час, то продолжительность рассчитывается в часах.
  • Гостей – количество гостей оформленных документом «Размещение».
  • Главный в группе – булево значение флажка отмечающего главного в группе, на которого будут произведены все начисления по группе. Для корректного отражения в отчете используйте «Формат» в таблице колонок.
  • Кол-во мест в номере – общее количество мест в номере, в который поселили гостя. Не зависит от размещения, зависит только от номера, в который гостя поселили.
  • Кол-во человек в номере – свойство номера, в который поселили гостя, отражает общее максимальное количество гостей, которое можно поселить в выбранный номер. Не зависит о размещения, зависит только от номера.
  • Номеров – количество номеров занятых данным гостем.
