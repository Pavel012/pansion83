Отчет "Шахматка"

Назначение

Имя отчета в конфигураторе «StateOfAllRooms».

Отчет выводит список всех номеров и по каждому номеру список проживающих гостей, резервирования, блокировки, которые действуют на указанную на форме отчета дату и время по указанным гостиницам, номерам и типам номеров.

Отборы на форме отчета

  • По типу номера - задает отбор по типу номера в гостинице.
  • По номеру - задает отбор по номеру, если поле оставить пустым, то будут выведены данные по всем номерам гостиницы.

Группировки

  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает на организацию, юр. лицо, от имени которого было произведено бронирование (размещение).
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Планируемый способ оплаты – ссылка на справочник «Способы оплаты». Разделитель учета, позволяющий указать, каким образом осуществляется оплата текущего гостя.
  • Номер – указание номера, в котором проживает гость.
  • Тип номера – ссылка на справочник «Типы номеров», указывает на тип номера, в котором размещен гость (планируется размещение).
  • Статус – ссылка на справочник «Статусы номеров», указывает статус номера, который обычно меняется при заезде и выселении из номера и обычно принимает значения «Занят», «Свободен», «Грязный», «Чистый».

Колонки

  • Вид размещения – ссылка на справочник «Виды размещений», указывает на вид текущего размещения.
  • Дата заезда – дата заезда в текущем размещении (брони).
  • Дата выезда – дата выезда в текущем размещении (брони).
  • Контактное лицо – ссылка на одноименное текстовое поле в документе «Бронь», указывает на лицо, бронировавшее данный номер от имени клиента. Обычно указывается при бронировании для контрагентов.
  • Документ – ссылка на документ, обосновывающий загрузку текущего номера. Это может быть «Размещение», «Бронь», или «Установка блокировки номера».
  • Гость – клиент, ссылка на справочник «Клиенты».
  • Это размещение – признак того, что текущая строка соответствует документу «Размещение».
  • Это блокировка – признак того, что текущая строка соответствует документу «Установка блокировки номера».
  • Это бронь – признак того, что текущая строка соответствует документу «Бронь».
  • Документ основание – в случае наличия основания у документа, указанного в колонке «Документ», выводится ссылка на этот документ.
  • Примечания – текстовый комментарий.
