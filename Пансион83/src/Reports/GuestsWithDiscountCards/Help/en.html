Отчет "Гости с дисконтными картами"

Назначение

Имя отчета в конфигураторе «GuestsWithDiscountCards».

Отчет выводит список действующих в указанном периоде дисконтных карт. Карта считается действующей если она не помечена на удаление и период действия карты пересекается с периодом отчета. Заблокированные карты в отчет попадают. Для того чтобы исключить заблокированные карты из отчета нужно открыть форму настроек отчета и добавить дополнительный отбор по полю "Карта заблокирована" установив условие "Равно" и значение "Нет".

Отборы на форме отчета

  • Период - задает отбор по периоду действия карт.
  • По типу скидки - задает отбор по типу блокировки в гостинице.
  • По типу клиента - задает отбор по типу номера в гостинице.
  • По гостинице - если поле оставить пустым, то будут выведены карты по всем гостиницам.

Группировки

  • Тип скидки - ссылка на тип скидки дисконтной карты.
  • Клиент – ссылка на справочник «Клиентов», клиент которому выдали карту.
  • Тип клиента – тип клиента, которому выдали карту.
  • Дисконтная карта - дисконтная карта.

Колонки

  • Дата начала периода действия дисконтной карты.
  • Дата окончания периода действия дисконтной карты.
  • Карта заблокирована – признак заблокированной карты.
  • Запрещены автоматические скидки - Если флажок включен, то после предъявления гостем дисконтной карты действует только скидка указанная в карте. При этом гость перестает участвовать в акциях и накопительных программах активных в данный момент в отеле.
  • Идентификатор карты - данные со "второй" дорожки магнитной карты, номер эмбоссированный на карте или UID RFID метки.
  • Количество - количество карт в отчете.
  • Примечания – Примечания к дисконтной карте.
