Отчет "Обороты по продажам ресурсов"

Назначение

Имя отчета в конфигураторе «ResourceSalesTurnovers».

Отчет позволяет получить итоги по продажам ресурсов в разрезе измерений регистра «Продажи ресурсов».

Отборы на форме отчета

  • По набору услуг - задает отбор по набору услуг.
  • По услуге - задает отбор по услуге, если поле оставить пустым, то будут выведенные данные по всем номерам гостиницы.
  • По ресурсу.
  • По типу ресурса.

Группировки

  • Учетная дата - дата услуги.
  • Учетный месяц - номер месяца услуги.
  • Учетный квартал - номер кватала услуги.
  • Учетная неделя - номер недели услуги.
  • Учетный год - год услуги.
  • Агент – ссылка на справочник контрагентов, указывает контрагента, который получает комиссию за гостя. Может также использоваться для аналитики работы агентов или туроператоров.
  • Тип клиента – характеристика гостя, может использоваться как разделитель учета для аналитики, также используется для задания разных цен по разным типам клиентов (каждому типу клиента можно поставить в соответствие свой тариф).
  • Фирма – юр.лицо гостиницы, от имени которой оформлено размещение гостя.
  • Договор – ссылка на договор контрагента, используется как разделитель учета взаиморасчетов с контрагентом.
  • Контрагент – ссылка на справочник контрагентов, указывает на организацию, юр. лицо, от имени которого было произведено бронирование (размещение).
  • Группа – объединяет несколько гостей, но может также состоять и из одного гостя. Каждый гость обязательно принадлежит какой–либо группе. Номер группы – это номер брони. В разрезе групп ведется учет взаиморасчетов с контрагентами.
  • Гостиница – название гостиницы, в которой размещен гость. Если в одной базе ведется учет по нескольким гостиницам, используется как разделитель учета.
  • Направление маркетинга – ссылка на элемент справочника «Направления маркетинга». Используется для анализа, может быть использован в группировках и отборах.
  • Документ основание - ссылка на документ «Бронь ресурса».
  • Отчетная валюта – валюта, указанная в качестве отчетной для гостиницы.
  • Ресурс – ссылка на справочник ресурсов, указывает на проданный ресурс.
  • Тип ресурса – указывает на тип забронированного ресурса.
  • Услуга – ссылка на справочник «Услуги».
  • Источник информации – ссылка на справочник «Источник информации», может использоваться для построения аналитических отчетов.

Колонки

  • Комиссия – сумма вычисленной комиссии для агента.
  • Комиссия без НДС – сумма вычисленной комиссии для агента за вычетом суммы НДС.
  • Сумма скидки – сумма предоставленной скидки.
  • Сумма скидки без НДС – сумма предоставленной скидки без учета НДС.
  • Продано часов работы ресурсов.
  • Количество – количество проданного ресурса.
  • Доходы от продажи ресурсов - сумма, на которую продано ресурсов в отчетный период.
  • Доходы от продажи ресурсов без НДС - та же сума без НДС.
  • Сумма – сумма продажи ресурса вместе с услугами, которые были начислены по брони.
  • Сумма продаж без НДС – фактическая сумма продажи ресурса и услуг за вычетом суммы НДС.
