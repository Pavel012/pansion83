Обработка "Заполнение цен по тарифам по дням"

Назначение

Обработка используется для заполнения таблицы цен по тарифам по видам размещения по дням "Цены по тарифам по дням". Таблица используется модулем он-лайн бронирования для быстрого получения цен и расчета стоимости возможных вариантов бронирования. Если таблица заранее рассчитанных цен не заполнена, то программа использует более медленный, но и более точный алгоритм. Использование этой обработки рекомендуется в случаях, когда не устраивает скорость поиска подходящих клиенту вариантов бронирования в модуле on-line бронирования на сайте гостиницы.

Относится к подсистеме: «Управление взаиморасчетами».

Способ вызова

Вызывается из меню "Обработки"
