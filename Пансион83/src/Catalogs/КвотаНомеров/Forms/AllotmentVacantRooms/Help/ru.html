Форма используется для работы с квотами номеров. В этой форме можно увидеть остатки свободных номеров по датам и по заездам. 

По гостиницам и типам номеров на каждую дату из периода отображения показываются 2 числа:

  • Сверху. Остаток свободных номеров/мест в квоте(ах)
  • Снизу зеленым цветом. Общее число свободных номеров/мест в гостинице (за вычетом номеров в квотах).

У каждого типа номера можно "плюсиком" открыть список квот. По каждой квоте на каждый день или заезд показывается остаток свободных номеров на 23:59:59. Если внизу формы на панели поиска включен флажок "Показывать бронь", то для каждой квоты "плюсиком" можно раскрыть карту бронирований/размещений, где прямоугольниками выделены периоды брони созданной по квоте и типу номера. Каждую бронь можно открыть двойным щелчком мыши на прямоугольнике.

Если на панели поиска включен флажок "Скрыть пустые", то те квоты и типы номеров, по которым на всем периоде формы нет свободных номеров, будут скрыты.

Внизу у формы располагается панель с доступными отборами. Отбирать данные можно по гостинице, тарифу, квоте, параметрам квоты, типу номера. Если в панели отборов указан агент/контрагент/договор, то на форме будут показаны остатки свободных номеров только по квотам выбранной организации. Если указан отбор по типу номера, то он действует не только на остатки по квотам, но и на остатки свободных номеров по всей гостинице. В полях отбора можно выбирать группы справочников.

На панели отбора расположены поля выбора периода отображения. На форме отображаются только те даты и заезды, которые попадают внутрь выбранного периода формы.

Справа на форме расположена панель операций. Эта панель позволяет быстро бронировать/селить гостей по периоду или заезду, добавлять/удалять номера в/из квоты, а так же обнулять квоту (т.е. сбрасывать остаток номеров/мест в квоте на 0). Обнулять квоту можно либо по выбранной категории номеров на выделенном в периоде, либо на всем периоде действия квоты. Период действия, тип номера, квота, тариф на этой панели устанавливаются автоматически при выделении области в таблице формы с датами. Форма позволяет выделив даты или заезд (в области с остатками свободных номеров) одним нажатием оформить по периоду бронь или размещение.

Флажок "Та же группа" позволяет последовательно выделяя периоды в форме добронировать их в выбранную группу номеров. При включении флажка поле "группа" заполняется номером последней забронированной группы.

Кнопка "Расчёт" позволяет (при условии, что в системе заведены шаблоны размещения) просмотреть стоимость брони по разным шаблонам. При выборе (подсвечивании) шаблона и нажатии кнопки "Забронировать" бронь сразу создается для всех видов размещений указанных в шаблоне. Если в колонке "N" заполнить кол-во номеров, то при бронировании будет открыта форма с количеством строк соответствующим выбранному шаблону и числу номеров.

Флажок "Показать документ" позвояет перед выполнением любой операции связанной с изменением состава квот сначала просмотреть документ, который программа автоматически создает для выполнения операции. При нажатии на кнопку "Бронь" при выключенном флажке программа сначала бронирует номер по указанной категории и только потом открывает карточку брони. Этот режим позволяет занимать свободный номер сразу в момент нажатия на кнопку "Бронь".

Графики заездов вводятся либо в карточке квоты, либо в отдельной форме, которая открывается по кнопке "Заезды" в панели инструментов формы списка квот.
